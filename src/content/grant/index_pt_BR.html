Concurso para Bolsa para Inovador do NetBeans


Os projetos dos sonhos se tornam realidade - Os ganhadores do concurso para bolsa para inovadores são anunciados!

29 de setembro de 2008

Após oito meses e mais de 170 propostas de projetos submetidas, nós apresentamos os ganhadores Concurso para Bolsa para Inovador do NetBeans!

Iniciado em Fevereiro de 2008, o Concurso para Bolsa para Inovador do NetBeans faz parte da iniciativa da Sun Microsystems em conceder 1 milhão de dólares para os desenvolvedores mundiais de código-fonte aberto, para que trabalhem em seu projeto favorito patrcionado pela Sun.

Ser pago para desenvolver um projeto dos sonhos ou em uma idéia que poderia ser de uso por outros desenvolvedores? A resposta da comunidade NetBeans foi imediata! Um painel de expertos do NetBeans Dream Team recebeu a incumbência de avaliar as melhores propostas. 20 projetos foram selecionados e seus desenvolvedores receberam a atribuição de mentores do NetBeans Dream Team e da Equipe de Engenharia do NetBeans. Após meses de desenvolvimento e feedback, 13 projetos foram completados, revisados pelo painel e aprovados para a concessão da bolsa.

Quatro projetos foram selecionados por alcançarem alto padrão de qualidade, usabilidade e que demonstraram potencial para crescimento futuro.

Citações sobre os projetos Medalha de Ouro e Prata por membros do NetBeans Dream Team e da Equipe de Engenharia do NetBeans

Sobre o Scala Language Support, Prêmio de Medalhista de Ouro

"O Scala Language Support é uma contribuição bastante complexa que fornece ao NetBeans IDE tudo que é necessário para desenvolver um aplicativo na linguagem de programação Scala".

Tomas Zezula, Engenheiro de Software da Infraestrutura Java

Sobre o Cube°n, Prêmio de Medalhista de Ouro

"O Cube°n traz o desenvolvimento focado em tarefa para o NetBeans 6.5. Ele é pesadamente influenciado pelo popular plug-in, Eclipse Mylyn, que alguns dizem ser o tremendo recurso do Eclipse. A primeira versão vem com suporte para o rastreador de questões Jira e há planos futuros para suportar o IssueZilla, Bugzilla e o XPlanner. Qualquer pessoa interessada neste plug-in é encorajada a instalar o mesmo e fornecer feedback para o desenvolvedor".

Ryan de Laplante, Membro do NetBeans Dream Team

Sobre o ivyBeans, Prêmio de Medalhista de Prata

"O IvyBeans traz o gerenciamento da dependência simples e rápido para projetos NetBeans com base no Ant. Ele realmente simplifica a tarefa de juntar um aplicativo a partir de repositórios de bibliotecas existentes. Isso é especialmente útil ao usar bibliotecas existentes de código-fonte aberto, mas pode simplificar a tarefa da mesma forma para qualquer projeto comercial. Encontre uma biblioteca, adicione-a ao seu projeto, e com ele todas as dependências estão imediatamente disponíveis para seu projeto sem qualquer outra configuração de projeto por parte do desenvolvedor. O que mais você poderia pedir?".

Wade Chandler, membro do NetBeans Dream Team

Sobre o PL/SQL Editor, Prêmio de Medalhista de Prata

"O PL/SQL Editor é um módulo pequeno mas com bons recursos que torna possível para que os que suportam a Oracle tenham um bom nível de suporte para sintaxe PL/SQL e o gerenciamento de objetos populares debtro de seu Java IDE favorito".

Masoud Kalali, membro do NetBeans Dream Team


"O PL/SQL Editor traz o desenvolvimento do banco de dados Oracle para o NetBeans. É fácil de usar e parece fornecer uma melhor experiência para os usuários Oracle dentro do NetBeans IDE. Ele ainda tem muito a evoluir, mas com o trabalho progredindo, use este projeto para tornar o desenvolvimento Oracle no NetBeans uma moleza. Ele parece muito promissor, e estou mirando adiante para seu futuro".

Wade Chandler, membro do NetBeans Dream Team
