Bolsa para Inovador do NetBeans - projetos selecionados

20 projetos selecionados para a Bolsa anunciados!

14 de abril de 2008

Projetos grandes

  • CashForward (Bill Snyder)
    CashForward é um aplicativo Swing para gerenciamento de dinheiro desenvolvido na plataforma NetBeans. Esperamos que o CashForward venha a fornecer um sólido exemplo de como usar a plataforma NetBeans Rich Client para aplicativos do mundo real.
  • Cube°n (Anuradha Gunasekara)
    Cube°n torna as tarefas parte como de primeira classe do NetBeans ao integrar com um sistema de rastreamento de questões. Ele então monitora sua atividade de trabalho para identificar o que é relevante e aprimora a produtividade ao reduzir a pesquisa, rolagem, navegação e expertise de compartilhamento. O suporte para integração com o JIRA está inicialmente planejado, mas a camada de abstração irá facilitar o suporte de outros repositórios como o Bugzilla e o Trac no futuro.
  • JavaSpaces (Magdalena Dukielska)
    Este projeto irá fornecer plug-ins para tornar fácil trabalhar com JavaSpaces no NetBeans -- até para iniciantes. Estes plug-ins irá permitir que os desenvolvedores se concentrem na lógica do programa, ao invés de lutar com a própria tecnologia JavaSpaces.
  • IvyBeans (Laurent Forêt)
    IvyBeans irá fornecer a integração completa do Apache Ivy com o recurso de gerenciamento de bibliotecas do NetBeans IDE. Isso inclui o suporte para gerenciar e baixar dependências, definir a classpath do projeto e editar os arquivo de configuração relevantes usados pelo Ivy.
  • NB Project (Alessandro Dourado)
    Este projeto irá criar um ferramenta de gerenciamento de projetos de software com base na plataforma NetBeans, usando um arquitetura cliente/servidor e um banco de dados para suportar o uso por desenvolvedores em uma equipe , assim como seu gerente. Ele irá suportar a definição de tarefa, gerenciamento de recursos e reporte de status do projeto.
  • NB-XUL (Aditya Kumar Sharma)
    Este projeto irá adicionar o suporte ao NetBeans para projetar interfaces de usuário usando a XML User Interface Language (XUL) do Mozzila. Este suporte irá incluir um XUL Editor com conclusão de código, realce de sintaxe, dobra de código e uma vista estrutural, assim como um depurador e assistentes que irão ajudá-lo a criar aplicativos XUL com facilidade e velocidade.
  • NetBeans Spot/Sun SPOT Plugin (Thiago Galbiatti Vespa)
    Este plug-in irá fornecer à Plataforma NetBeans a habilidade de integrar a interface de acesso Sun Spot através de uma rede sem fio, um gateway, ou ao usar o emulador Sun Spot.
  • NbPython (Allan Davis)
    O nbPython irá adicionar o suporte de primeira classe para a linguagem Python ao NetBeans IDE, similar a como o IDE já suporta a linguagem Ruby.
  • Scala Support (Caoyuan Deng)
    Este projeto irá fornecer um ambiente de desenvolvimento integrado para compilar, executar, testar e depurar aplicativos Scala.
  • Visual JavaFX (Adam KÄ™dziora)
    O objetivo deste projeto é o de criar um editor visual para o JavaFX, completo com ferramentas de desenho com base em vetores, suporte para gerar applets JavaFX , criação de animações, e mais.

Projetos pequenos

  • CoffeeDregs (Kees Huizing)
    O CoffeeDregs é uma ferramenta que visualiza passo a passo a execução de um programa Java arbitrário. Esta ferramenta está dirigida aos estudantes e os permitirá passar linha por linha de seu código para a execução visual do código. Diferente de um depurador, esta ferramenta faz a renderização visual dos objetos, incluindo campos e chamadas de métodos no momento ativos.
  • JSpree (ManiKanta G)
    Este módulo irá fornecer uma biblioteca de referência completa para o JavaScript, incluindo os detalhes de implementações do navegador de todos os objetos e métodos. Ele também irá suportar a adição de snippets de código, permitindo que um desenvolvedor arraste e solte seus snippets em um editor.
  • Netbeans Update Service (Mark Ashworth)
    Este projeto tem o objetivo de tornar fácil configurar um servidor central para espelhar o Centro de Atualização do NetBeans em uma rede corporativa. Isso irá permitir que os desenvolvedores sem acesso à Internet, mantenham seus IDEs atualizados enquanto economiza o uso de banda larga.
  • MONOH (Carlos Oliveira)
    O MONOH irá adicionar reposta por voz ao NetBeans IDE. Isso irá permitir que os usuários o configurem para reconhecer uma palavra ou até uma frase completa que irá acionar uma ação. Essa ação poderá abrir um projeto, executar um aplicativo ou até criar uma classe Java.
  • Numbered Bookmarks (M.A.S. Jayasundara)
    O plug-in Numbered Bookmarks irá permitir que os usuários referenciem bookmarks em seus arquivos por número, ao invés de ter que passar por eles para encontrar um, e irá suportar a navegação entre bookmarks em diferentes arquivos. Os usuários serão capazes de alternar entre o comportamento padrão e Bookmarks Numerados.
  • PDFViewer (Steve Tzou)
    Este projeto irá usar a biblioteca PDF Renderer da Sun para adicionar suporte incorporado para visualizar documentos PDF dentro do NetBeans IDE.
  • PL/SQL Editor (Alexandre Soumbatov)
    Este projeto irá adicionar o suporte para editar o Oracle PL/SQL dentro do NetBeans IDE, incluindo o realce de sintaxe, dobra de código e identação.
  • Project Darkstar Tools and Mobility Support (Karel Herink)
    O Project Darkstar tem o grande potencial de revolucionar de forma massiva o desenvolvimento de jogos on-line multi-player, e nós intencionamos tonar mais fácil seu desenvolvimento ao criar um plug-in para o NetBeans IDE.
  • Regular Expressions Module (Angad Singh)
    Este projeto objetiva criar uma solução completa simples e fácil para ajudar os desenvolvedores em todos os aspectos de trabalhar com expressões regulares no NetBeans, incluindo o aprendizado, compilação, teste, depuração, visualização, perfilamento, análise, organização, pesquisa e compartilhamento de expressões regulares.
  • Resource Bundle Editor (Denis Stepanov)
    Este projeto irá adicionar o suporte aprimorado para editar conjuntos de recursos no NetBeans IDE, incluindo as propriedades de classificação, agrupamento e pesquisa, assim como a detecção de propriedades não utilizadas.



Parabéns a todos os proprietários de projeto que forma selecionados para o concurso da bolsa, e muito obrigado a todos que participaram.
