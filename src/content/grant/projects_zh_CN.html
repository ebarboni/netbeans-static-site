NetBeans 创新人员奖励基金 - 精选的项目

公布了为奖励基金选出的 20 个项目！

2008 年 4 月 14 日

大型项目

  • CashForward (Bill Snyder)
    CashForward 是在 NetBeans 平台上构建的资金管理 Swing 应用程序。我们预计 CashForward 会提供切实可行的示例，以说明如何使用 NetBeans 富客户端平台构建实际应用程序。
  • Cube°n (Anuradha Gunasekara)
    Cube°n 通过与问题跟踪系统集成一起，使任务成为 NetBeans 的重要组成部分。然后，它监视您的工作活动以确定相关问题，并通过减少搜索、滚动、导航和共享专业知识来提高工作效率。最初计划了与 JIRA 的集成支持，但抽象层有助于将来支持其他资源库，如 Bugzilla 和 Trac。
  • JavaSpaces (Magdalena Dukielska)
    此项目提供了一些插件以简化在 NetBeans 中使用 JavaSpaces 的过程，甚至对初学者也是如此。通过使用这些插件，开发者可以将重点放在程序逻辑上，而不是修补 JavaSpaces 技术本身。
  • IvyBeans (Laurent Forêt)
    IvyBeans 将 Apache Ivy 完全集成到 NetBeans IDE 库管理功能中。这包括支持管理和下载依赖关系、设置项目类路径以及编辑 Ivy 使用的相关配置文件。
  • NB 项目 (Alessandro Dourado)
    此项目使用客户端/服务器体系结构和数据库创建基于 NetBeans 平台的软件项目管理工具，以支持团队开发者及其经理使用该工具。它支持任务定义、资源管理和项目状态报告。
  • NB-XUL (Aditya Kumar Sharma)
    此项目将添加对 NetBeans 的支持，以使用 Mozilla 的 XML 用户界面语言 (XUL) 设计用户界面。这种支持包括具有代码完成、语法突出显示、代码折叠和结构视图的 XUL 编辑器以及帮助您轻松快捷地创建 XUL 应用程序的调试器和向导。
  • NetBeans Spot/Sun SPOT 插件 (Thiago Galbiatti Vespa)
    此插件为 NetBeans 平台提供了通过无线网络、网关或使用 Sun Spot 仿真器与 Sun Spot 访问接口集成在一起的功能。
  • NbPython (Allan Davis)
    nbPython 在 NetBeans IDE 中添加了重要 Python 语言支持，这与 IDE 支持 Ruby 语言的方式类似。
  • Scala 支持 (Caoyuan Deng)
    此项目提供一个集成开发环境以构建、运行、测试和调试 Scala 应用程序。
  • 可视 JavaFX (Adam Kędziora)
    此项目的目标是为 JavaFX 创建可视编辑器、基于矢量的绘图工具以及支持生成 JavaFX Applet 和创建动画等。

小型项目

  • CoffeeDregs (Kees Huizing)
    CoffeeDregs 是一个工具，用于逐步可视化任意 Java 程序的执行过程。此工具针对的是学生，他们可以逐行执行代码以及可视化程序的执行过程。与调试器相比，此工具以可视方式呈现对象，包括字段和当前活动方法调用。
  • JSpree (ManiKanta G)
    此模块为 JavaScript 提供完整的参考库，包括所有对象和方法的浏览器实现细节。它还支持添加定制代码片段，以使开发者直接将其片段拖放至编辑器中。
  • Netbeans 更新服务 (Mark Ashworth)
    此项目旨在简化设置中心服务器以在企业网络上镜像 NetBeans 更新中心的过程。这样，无法访问 Internet 的开发者也可以及时更新 IDE，同时还会节省带宽。
  • MONOH (Carlos Oliveira)
    MONOH 在 NetBeans IDE 中添加语音应答。这样，用户就可以对其进行配置，以识别将触发操作的字词或甚至整个句子。该操作可能会打开项目，运行应用程序或甚至创建新的 Java 类。
  • 编号的书签 (M.A.S. Jayasundara)
    通过使用编号的书签插件，用户可以在文件中按编号引用书签，而无需循环访问文件以查找书签，并且支持在不同文件中的书签之间进行导航。用户可以在缺省行为和编号的书签之间进行切换。
  • PDF 查看器 (Steve Tzou)
    此项目使用 Sun 的 PDF 呈现器库添加直接在 NetBeans IDE 中查看 PDF 文档的内置支持。
  • PL/SQL 编辑器 (Alexandre Soumbatov)
    此项目添加了直接在 NetBeans IDE 中编辑 Oracle PL/SQL 的支持，包括语法突出显示、代码折叠和缩进。
  • Project Darkstar 工具和 Mobility 支持 (Karel Herink)
    Project Darkstar 在变革大型多玩家在线游戏开发方面有很大的潜力，我们打算通过为 NetBeans IDE 创建插件来简化开发过程。
  • 正则表达式模块 (Angad Singh)
    此项目旨在创建一个简单且易于使用的综合解决方案，帮助开发者完成在 NetBeans 中使用正则表达式的所有工作，其中包括了解、构建、测试、调试、可视化、分析、组织、搜索和共享正则表达式以及分析性能。
  • 资源包编辑器 (Denis Stepanov)
    此项目添加了在 NetBeans IDE 中编辑资源包的改进支持，其中包括排序、分组、搜索属性以及检测未使用的属性。
