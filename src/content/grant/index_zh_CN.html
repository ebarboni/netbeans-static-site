NetBeans 创新人员奖励基金比赛


将梦想项目变成现实 - 公布了创新人员奖励基金比赛获奖者！

2008 年 9 月 29 日

历时 8 个月，我们共收到 170 多个提交的项目，并从中评出了 NetBeans 创新奖励基金比赛获奖者！

2008 年 2 月，我们举办了 NetBeans 创新奖励基金比赛，这是 Sun Microsystems 计划的一部分，旨在向世界各地的开源开发者提供 100 万美元以完成他们喜爱的 Sun 资助项目。

您希望获得资助以开发可能对其他开发者非常有用的梦想项目或创新作品吗？NetBeans 社区会立即作出答复！来自 NetBeans 梦之队的专家评审团的任务是评估最好的建议书。共有 20 个项目入选，并为这些项目的开发者分配来自 NetBeans 梦之队和 NetBeans 工程团队的导师。经过几个月的开发和反馈，评审团完成并审查了 13 个项目并批准提供奖励基金。

获奖项目：

大型项目
  • CashForward，Bill Snyder（11,500 美元）
  • Cube°n，Anuradha Gunasekara（11,500 美元）
  • JavaSpaces，Magdalena Dukielska（11,500 美元）
  • IvyBeans，Laurent Forêt（11,500 美元）
  • NB 项目，Alessandro Dourado（11,500 美元）
  • NetBeans Spot/Sun SPOT 插件，Thiago Galbiatti Vespa（11,500 美元）
  • Scala 支持，Hong Wei Deng（11,500 美元）
小型项目
  • MONOH，Carlos Oliveira（2,000 美元）
  • 编号的书签，Samitha Jayasundara（2,000 美元）
  • PDF 查看器，ZhenYa Zou（2,000 美元）
  • PL/SQL 编辑器，Alexandre Soumbatov（2,000 美元）
  • Project Darkstar 工具和 Mobility 支持，Karel Herink（2,000 美元）
  • 资源包编辑器，Denis Stepanov（2,000 美元）

 

有 4 个项目由于满足较高的质量标准和可用性并具有未来增长潜力而入选。

金奖得主

  • Scala 支持（11,000 美元）
  • Cube°n（11,000 美元）

银奖得主

  • IvyBeans（5,000 美元）
  • PL/SQL 编辑器（5,000 美元）

NetBeans 梦之队和 NetBeans 工程团队成员对金奖和银奖项目的评语

关于 Scala 语言支持，金奖得主

“Scala 语言支持是一个非常复杂的项目，它为 NetBeans IDE 提供了使用 Scala 编程语言开发应用程序所需的所有功能。”

NetBeans Java 基础结构软件工程师 Tomas Zezula

关于 Cube°n，金奖得主

“Cube°n 将侧重于任务的开发应用于 NetBeans 6.5。这深受常用 Eclipse Mylyn 插件的影响，人们将其看作 Eclipse 的替代功能。第一个版本提供了 JIRA 问题跟踪器支持，将来还会计划支持 IssueZilla、Bugzilla 和 XPlanner。建议感兴趣的人员安装该插件并向开发者提供反馈。”

NetBeans 梦之队成员 Ryan de Laplante

关于 ivyBeans，银奖得主

“IvyBeans 为基于 Ant 的 NetBeans 项目提供了简单快速的项目依赖关系管理功能。它真正简化了通过现有的库资源库组成应用程序的任务。这在使用常用的开源库时特别有用，但也可以简化任何商业项目的任务。查找一个库，将其添加到您的项目中，可以在您的项目中立即使用该库以及所有依赖关系，开发者无需进行其他项目配置。除此之外，您还能有什么别的需要呢？”

NetBeans 梦之队成员 Wade Chandler

关于 PL/SQL 编辑器，银奖得主

“PL/SQL 编辑器是一个很小但功能完善的模块，Oracle 倡导者可以在其喜爱的 Java IDE 中获得良好的 PL/SQL 语法和常用对象管理支持。”

NetBeans 梦之队成员 Masoud Kalali


“PL/SQL 编辑器将 Oracle 数据库开发应用于 NetBeans。它易于使用，并在 NetBeans IDE 中为 Oracle 用户提供更好的体验。该项目还有很多地方需要完善，但随着工作的进展，此项目可以使 NetBeans 中的 Oracle 开发工作变得更加轻松。该项目非常有前途，我对它的未来充满信心。”

NetBeans 梦之队成员 Wade Chandler
