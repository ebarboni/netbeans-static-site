歡迎來到 NetBeans 和 www.netbeans.org

您好，歡迎來到 NetBeans 的家，這是一個開放原始碼的 Java 整合開發環境。

什麼是 NetBeans？

NetBeans 是一個成功的開放原始碼計畫，擁有全球近 100 個合作伙伴不斷發展茁壯的社群團體。Sun Microsystems 於 2000 年 6 月，發佈了 NetBeans 開放原始碼計畫，並繼續為該計畫的主要贊助商。

目前存在兩項產品: NetBeans IDE 和 NetBeans Platform。

NetBeans IDE 是一個開發環境 － 供程式設計師撰寫、編譯、除錯和部署程式的一個工具。他採用 Java 編寫而成 － 但能夠支援各種程式語言。另外，也有相當龐大的模組來擴充 NetBeans IDE。他是一個免費產品，不限制其使用形式。

此外還可以獲得 NetBeans Platform; 一個模組化的可擴充基礎，用於發展大型桌面應用程式的軟體基礎。ISV 合作伙伴提供加值的模組套件，他們很容易整合到 IDE 中，並且也可用來開發其本身的工具及解決方案。

兩項產品均採用開放原始碼模式，並免費供商業及非商業行為使用。原始碼部分，採用 Common Development and Distribution License (CDDL).授權條款。

什麼是 netbeans.org？

netbeans.org 是 NetBeans 開放源碼團體之家，它們都共同致力於建立世界級的 IDE。 netbeans.org 允許來自全球 160 多個國家的使用者連接到 NetBeans 周圍的資源和人們。在這裡，您可以下載最新的 NetBeans 版本，查看獻上支援文件，建立您的個人 Java 知識庫，追蹤新聞，加入郵件論壇清單，奉獻源原始碼，了解與計劃相關的人們，互相交流，等等等等。
