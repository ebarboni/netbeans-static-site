欢迎使用 NetBeans 项目！

1. 什么是 NetBeans 项目？

NetBeans IDE 是一个开源集成开发环境，这是可以免费下载和使用以进行应用程序开发的软件工具。更多信息...。

NetBeans 平台是一个可扩展的框架，您可以使用该框架构建与操作系统无关的专业桌面应用程序。更多信息...。

最后，NetBeans.org 是一个蓬勃发展的开源社区，其中汇聚了世界各地的数十万个经验丰富的开发者。更多信息...。

2. 我如何开始使用呢？

下载 NetBeans IDE。它具有开发世界一流的应用程序所需的所有功能。NetBeans 软件是 100% 基于 Java 的 IDE 和平台，因此，它可以在每个具有 JDK 的操作系统中使用。NetBeans 支持各种不同的 Java 技术，但并不限于 Java，它还支持其他语言。NetBeans 工具具有非常多的功能，您可以选择最适合您的下载内容。在功能页面上了解支持的技术和功能的详细信息。您可以使用插件模块添加更多功能（在 IDE 中，选择“工具”>“插件”）。

3. 我可以在哪里参与开发？

这是一个蓬勃发展的 NetBeans 用户社区，不仅拥有很多新用户，而且还拥有一些技术专家。请与他们一起讨论提示和技巧、NetBeans 功能、问题解决，等等。立即注册！

4. 我如何了解更多信息？

netbeans.org 上提供了丰富的知识，您可以认真学习这些内容，而且每周还会发布新的内容。教程、文章、指南、演示和截屏视频 - 全都包含在内。
