1. NetBeans プロジェクトとは

NetBeans IDE は、オープンソース統合開発環境で、アプリケーションの開発のためにダウンロードして、無料で使えるソフトウェアツールです。詳細...

NetBeans Platform は、プロ向けの OS に依存しないデスクトップアプリケーションを構築するために使用できる拡張可能なフレームワークです。詳細...

最後に、NetBeans.org は世界各地の数十万人の経験豊富な開発者が集う活気あるオープンソースコミュニティーです。詳細...

2. どのようにして始めるのですか。

NetBeans IDE をダウンロードしてください。世界クラスのアプリケーションの開発に必要なすべてのものが揃っています。NetBeans ソフトウェアは 100% Java ベースの IDE およびプラットフォームであるため、JDK を使用できるすべての OS で動作します。NetBeans は幅広い Java テクノロジをサポートしますが、Java に限定されず、その他の言語もサポートしています。NetBeans ツールにはきわめて多くの機能が備えられているため、最適なものを選んでダウンロードすることができます。サポートされるテクノロジと機能については、機能ページを参照してください。プラグインモジュールによって機能を追加できます (IDE で、「ツール」>「プラグイン」を選択します)。
