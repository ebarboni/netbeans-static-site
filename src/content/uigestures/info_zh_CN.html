什么是 UI 操作收集器？

这是查看 NetBeans 全面统计信息的入口页面。UI 操作收集器作为一个门户，可用来查找有关 NetBeans 使用方法的一些新提示，了解使用相同技术的开发者在哪些方面执行的 UI 操作最多，并可以向 NetBeans 开发团队提交您的反馈。

它的基本思想在于：通过该门户，让服务器了解您的操作，服务器随后会根据接收的数据生成一些度量标准、统计信息，或直接向您提供一些针对性建议，以便您可以采用更好的方式来使用 NetBeans IDE。

如今，您可以通过该门户提交有关截至目前在 IDE 中所执行的 UI 操作的数据，只要您如此执行，服务器便会记录这些数据并向您提供一些统计信息。我们可以提供鼠标和键盘平均使用情况、代码完成、常用操作的频率等信息。

Please do not wait! Submit your data to help us make NetBeans better. You are welcomed to see and inspect the data before the transfer by clicking View Data button. If you like the system, you can also turn on an automatic mode when the data will be transfered to the server everytime the local buffer is full.
