NetBeans Glyphs

Not all Java IDE´s are created equal and the launch of NetBeans IDE 5.0 proves this beyond any reasonable doubt. With its release, we´re ushering in a new era for developers; one that values functionality and productivity yet, at the same time, supports innovation and collaboration.

We call this essence "Systematic Openness" and it is the spirit that permeates all things NetBeans.

The graphics below are a celebration of that spirit. Each glyph represents the number 5 in a different language. After all, what´s more systematic than numbers, and, what´s more open than communicating across cultures?

This page is just a starting point.

You have not seen the last of these glyphs nor the philosophies they represent.
