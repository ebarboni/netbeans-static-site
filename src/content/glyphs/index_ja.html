NetBeans グリフ

すべての Java IDE が等しく作成されているわけではなく、NetBeans IDE 5.0 の立ち上げは、あらゆる合理的疑いを超えてこれを証明しています。そのリリースに伴い、私たちは開発者のための新しい時代を迎え入れようとしています。機能性や生産性をまだ重視しながら、同時に革新や協調もサポートしていく時代です。

この根本精神は「Systematic Openness (系統だったオープン性)」と呼ばれ、この精神は NetBeans のすべてのものに浸透しています。

下のグラフィックスは、その精神を記念するものです。各グリフはさまざまな言語での数字 5 を表しています。結局のところ、数字よりも系統的なものは何ですか、また、文化を超えてコミュニケーションを取るよりもオープンなものは何ですか。

このページは出発点にすぎません。

これらのグリフの最後もそれらが表す哲学もまだ見ていません。
