NetBeans Multilingual (ML) DVD Starter Kit プログラム

NetBeans Multilingual DVD Starter Kit に関心をお持ちいただき、ありがとうございます。2005 年 8 月以来、NetBeans.org では、Starter Kit プログラムを通じて世界中の開発者に 225,000 を超える無料の CD および DVD を出荷してきました。

やがて、NetBeans はコミュニティーとして劇的な成長を遂げ、それに伴ってソフトウェアの需要も増しました。

今後は NetBeans DVD Starter Kit の送料は無料になりませんが、受賞したツールをコミュニティーで引き続き受け取るいくつかの方法を提供します。

  • 23M バイトほどの軽量のインストール無料バンドルについては、NetBeans IDE のダウンロードのページを参照してください。IDE にはあとでいつでもより多くの機能を追加できます。

  • NetBeans IDE Multilingual (ML) DVD Starter Kit の無料の ISO イメージをダウンロードして書き込みます。

  • ダウンロードしたファイルの一時停止、再開、再起動、および検証が行えるようになる Oracle の JavaSE Download Manager を使用して NetBeans IDE + JDK バンドルをダウンロードします。

引き続き変わらぬご支援と、NetBeans プロジェクトに関心をお持ちいただいたことにあらためてお礼申し上げます。NetBeans チームと意見や質問を共有し、メーリングリストやフォーラムに参加して、コミュニティーに貢献する方法を模索されることを歓迎し奨励します。
