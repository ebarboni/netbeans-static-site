Contact the NetBeans Team

There are a few ways to get in touch with the NetBeans Team. Please read this page carefully to make sure to direct your query to the appropriate channel.

Support and Use of NetBeans

Please direct all questions regarding use and support of the NetBeans IDE and Platform to the public nbusers@netbeans.org mailing list or forum. You must subscribe to the list/forum to post questions.

NetBeans Support Forums and Mailing Lists
NetBeans Support Page

Bug Reports, Feature Requests and Enhancements

To report a bug in the NetBeans IDE, please file an issue. If you have feature or enhancement requests, we recommend that you first ask on the nbusers mailing list to gauge if your idea is feasible, supported by others, and if another developer isn't already working on it.

Search existing issues to see if your issue has already been reported

File a new bug report or enhancement request

News

Have you released a NetBeans plugin or a NetBeans Platform-based application? Heard of a conference or event that could be of interest to NetBeans users? Submit your news to the NetBeans Newsletter!

Submit a news item to the NetBeans Weekly editors

Webmaster

For problems with your NetBeans.org account; broken downloads or links; web images or pages not displaying properly; issues with website content; or suggestions for website improvements:

Contact the NetBeans Webmaster
