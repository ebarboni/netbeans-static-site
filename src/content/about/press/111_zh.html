SUN MICROSYSTEMS LAUNCHES PREMIER OPEN SOURCE AND COMMERCIAL JAVA(TM) TOOLS FOR CHINESE DEVELOPERS

[新闻稿]

2003年10月16日

Sun 公司发布主流开放源码和商用 Java 工具中文版

（ SUN MICROSYSTEMS LAUNCHES PREMIER OPEN SOURCE AND COMMERCIAL JAVA(TM) TOOLS FOR CHINESE DEVELOPERS ）

中国开发人员可借助功能完备的开放源码 Java IDE 进行桌面及Web应用开发

J2EE使开发人员拥有完备的Web服务技术和企业级开发环境

(Chinese developers enabled with a full-featured Open Source Java IDE for desktop and web applications; J2EE focused developers enabled with a comprehensive Web Services and Enterprise Tier development environment)

（2003 年10 月16 日讯） 今天， Sun Microsystems 公司在北京召开的“第六届全国 Java 技术与应用大会”上发布了中文简体版 NetBeans 3.5.1 版本和 Sun Java Studio 5 (原 Sun ONE Studio 5) 标准版产品。 NetBeans 是开放源码的 Java 集成开发环境 (IDE) ，适用于各种客户机和Web应用。现在，用户可在开放源代码库和社区网站netbeans.org免费下载 NetBeans IDE (网址https://netbeans.org/index_zh_CN.html)。 Sun Java Studio 是 Sun 公司最新发布的商用全功能 Java IDE ，支持 Solaris 、 Linux 和 Windows 平台，适于创建和部署 2 层 Java Web 应用和n层 J2EE 应用的企业开发人员使用。 此次面向中国市场推出的这两款新的工具令人振奋。它们的推出将会极大地简化 Java 应用开发，提高开发工作效率，发挥支持创新并加快产品上市速度的重要作用。

Sun 公司负责开发者平台项目的副总裁 Rich Green 表示：“ Java 技术再一次证明了它孕育新市场的潜力，为 Java 开发人员、销售商和他们的客户创造了新的机遇。在异构环境中应用互操作性与便携性领域，Java技术一直傲视群雄。今天这两款新产品的发布表明， Sun 正在使 Java 成为比以往任何时候都更加易于使用的技术，并且让 Java 能够面向最广泛的开发群体，从企业开发人员到独立开发商，他们往往采用开放源技术或从事大型开发与支持项目，或致力于 Sun 一直给予积极支持的商用环境，从而更加强化 Java 技术在网络计算平台中的优选地位。”

Web 应用开发： NetBeans 3.5.1
NetBeans 是一个全功能的开放源码 Java IDE ，可以帮助开发人员编写、编译、调试和部署 Java 应用，并将版本控制和XML编辑融入其众多功能之中。 NetBeans 3.5.1 可支持 Java 2平台标准版(J2SE)应用的创建、采用 JSP 和 Servlet 的 2 层 Web 应用的创建，以及用于2层 Web 应用的 API 及软件的核心组的创建。此外， NetBeans 3.5.1 还预装了一个 Web 服务器，即 TomCat ，从而免除了繁琐的配置和安装过程。所有这些都为 Jav a开发人员创造了一个可扩展的开放源多平台的 Java IDE ，以支持他们在各自所选择的环境中从事开发工作，如 Solaris 、 Linux 、 Windows 或 Macintosh 。 NetBeans 3.5.1 也为面向 JDK 1.4.0 、 1.4.1 和 1.4.2 的应用开发提供支持。

Sun 公司 NetBeans.org 总监 Dave Sundstrom 表示：“ NetBeans 是业界第一款支持创新型Java开发的开放源码IDE。基于NetBeans在业界的领导地位，并且已经十分成熟，所以，现在 NetBeans 是唯一面向所有平台开发人员提供的全兼容的 Java 开放源码 IDE 。Sun公司始终支持开放源码社区并积极参与其中。通过这一中文版产品的发布，中国的开发人员可以利用业界强大的开发工具来构建桌面、 Web 或移动应用，且使自己融入全新的 Java 创新境界。”

通过 NetBeans 和开放的 API 的模块化结构，第三方能够非常轻松地扩展或集成NetBeans平台。全世界已有将近 100 个公司及其开发人员加入到 NetBeans 性能扩展的行列，而且这种势头还在迅速增长。全球对 NetBeans 的下载次数已经超过了 170 万。

NetBeans.org 理事会成员 Maxym Mykhalchuk 指出：“ NetBeans 开放源项目为越来越多国家的开发人员进行跨平台开发提供了全兼容的 Java IDE 和应用架构。我们对 Sun 在支持开放源码社区和投资 NetBeans 研发所取得的成绩表示祝贺。从 NetBeans 3.5.1 版本开始，中国的开发人员就拥有了一个免费的、开放的、经过严格测试并得到官方支持的本地化开发工具，并且可以加入业已为全世界英语、日语和俄语国家开发人员建立起的庞大的开放源码社区之中。”

作为 NetBeans 代码库的创始成员和杰出贡献者， Sun 还推出了冠以 Sun 品牌的产品—— Sun Java Studio 5 标准版，这对基于此种架构进行企业级应用开发提供了特殊的附加价值。 企

业开发： Sun Java Studio 5
Sun Java Studio 5 标准版适用于那些创建和部署2层 Java Web 应用和n层 J2EE 应用的企业开发人员。新近推出的 IDE 与 Sun Java Application Server 7 软件紧密集成， IDE 是 Sun Java Application Server 7 安装时默认的运行环境，允许一键式动态部署与执行 J2EE 应用。这款产品的特色还表现在，拥有附加的 JAX-RPC （ Java to XML - Remote Procedure Call） Web 服务 API ，支持安全、附件和句柄，启动时间和环境响应速度两方面的性能都得到显著改善；通过 Java Web 服务开发者包1.0提供 Web 开发支持；包括流行的 JUnit Java 测试软件。此外，该产品在提供部署和移植服务的同时，还为开发人员提供免费支持与培训。

今年晚些时候， Sun 公司还将推出加强型 Sun Java Studio 5 标准版，致力于加速基于 Java 技术的 Web 层应用开发。这一最新的创新性举措将为企业开发人员提供更高水平的功能集合，从而迅速、有效、轻松地引导他们熟悉不断发展的 J2EE 技术规范、工业设计模式和最佳实践案例，有能力开发出在复杂度、可缩放性和成熟度等方面要求都满足复杂的企业应用需求的 Java 代码。

InsiTech 集团公司总裁兼 CTO David Pociu 表示：“在我们看来， Sun Java Studio 是目前最好的 IDE ，对于 Java 平台更是如此。同时， Sun Java Studio 还是最开放的平台，可以轻松地集成插入式模块，使用起来极其方便。”

Sun Java Studio 5 标准版是前一企业版的完全替代产品，而售价大约只是以前版本的50%，价格优惠程度也十分可观。更多关于 Sun Java Studio 5 的信息请参见 http://www.sun.com/software/sundev/jde。

为这些业界领先的 Java 开发环境推出简体中文版，再一次证明了 Sun 公司为帮助不同水平层次和产业细分市场的中国开发人员构建强大的综合性 Java 应用而付出的努力。凭借这个使用方便、功能丰富并且在价格上极具竞争力的软件专业平台， Sun 公司赋予开发人员更强大的开发能力。

NetBeans.org 简介
NetBeans.org 由 Sun 公司在 2000 年创立，它是开放源运动以及开发人员和用户社区的家园，旨在构建世界级的 Java IDE 。 NetBeans.org 全球 IDE 下载次数已超过 160 万，拥有2万多个注册成员，并且还在不断发展壮大。其最新研发的 NetBeans 3.5.1 中文版将帮助中国的开发人员创建更迅速、更稳定和更灵活的开发环境。 NetBeans 目前可以在 Solaris 、 Windows 、 Linux 和 Macintosh OS X 平台上进行开发，并在 SPL ( Sun 公用许可) 范围内使用。 NetBeans 3.5.1 和开放源码网站www.netbeans.org已经获得业界广泛认可，并支持 NetBeans 扩展 IDE 模块目录中大约100多个模块。更多信息请参见 http://www.netbeans.org。
