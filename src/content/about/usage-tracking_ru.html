Отслеживание данных об использовании NetBeans

Цель отслеживания данных об использовании состоит в сборе и анализе статистических данных об использовании пользователями высокоуровневых функциональных возможностей NetBeans. Более точное представление о том, какие функции являются наиболее и наименее используемыми помогает в планировании будущих выпусков. Собранные данные строго анонимны и отправляются на сервер netbeans.org автоматически. Сбор и отправка данных не влияет на производительность и на обычный режим работы.

Типы собираемых данных об использовании

  • Конфигурация среды IDE: номер сборки, информация об ОС, список включенных подключаемых модулей
  • Использование типов проектов (например, Java SE, Java SE Freeform, Веб-приложение Java, Приложение Java Enterprise, Приложение Ruby/Rails, PHP, C/C++, Проект модуля NB и т.д.)
  • Использование платформ веб-приложений (например, Struts, JSF, Visual JSF, Spring, Hibernate и т.д.)
  • Типы и версии серверов и сред выполнения, используемых для развертывания (например, GlassFish, Tomcat, WebSphere, JBoss, WebLogic, WEBrick и т.д.)
  • Типы драйверов JDBC, используемых для подключения к базам данных (например, MySQL, JavaDB, Oracle и т.д.)
  • Использование конкретных функциональных возможностей по повышению производительности труда (например, отладчика, профайлера, GUI Builder)
  • Типы систем управления версиями, используемые пользователями (CVS, Subversion, Mercurial, ClearCase и т.д.)
  • Обращение к темам справки.

Типы НЕ собираемых данных об использовании

  • Имена пользователей и пароли.
  • Пути, имена файлов и классов.
  • Фрагменты исходного кода пользователей.

Группа NetBeans сердечно благодарит за участие в программе отслеживания данных об использовании!
