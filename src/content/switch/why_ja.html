NetBeans IDE に切り替える主な理由


NetBeans 統合開発環境 (IDE) で計画を実現します。NetBeans IDE は、Java SE、Java EE、または Java ME テクノロジに加え、PHP、Groovy、JavaScript、および C/C++ を操作する場合の生産性を向上できます。スケルトンコードを生成するビジュアルツールも使用できるため、コードを 1 行も書くことなく基本的なアプリケーションを作成できます。

次に NetBeans IDE を使用する主な理由を示します。
  1. すぐに動作する

    NetBeans IDE をダウンロードして、インストールするだけで十分です。インストールは小さなダウンロードサイズでとても簡単です。すべての IDE ツールと機能は完全に統合されており、プラグインを探す必要がなく、IDE を起動すると、それらが連携します。

  2. 無料のオープンソース

    NetBeans IDE を使用すると、数千のユーザーが助け合い、貢献する用意ができている活気のあるオープンソースコミュニティーに参加します。NetBeans メーリングリストでの意見交換、PlanetNetBeans のブログ、役立つ FAQ があります。

  3. Connected Developer

    NetBeans IDE はコラボレーション環境で作業するチームに選ばれるツールです。Jira と Bugzilla の両方を使用したファイル課題追跡レポートなどの java.net でホストされるプロジェクトを作成し、管理でき、さらに同じ考えを持つ開発者全員が、使い慣れた IDE のインタフェース内から直接にコラボレーションできます。

  4. 強力な GUI ビルダー

    GUI ビルダー (以前は Project Matisse と呼ばれていた) は高度でありながら簡単な Swing Application Framework と Beans Binding をサポートしています。自然な方法で GUI を構築できるようになりました。

  5. Java 標準規格およびプラットフォームのサポート

    IDE は、最新の Java 標準規格を含むすべての Java 開発プラットフォームのエンドツーエンドソリューションを提供します。

    • Java Mobility のサポート: モバイルデバイス用アプリケーションの作成、テスト、実行のための完全な環境。プリプロセッサブロックによって、断片化問題を簡単に処理できます。Java Mobility 開発のサポートは、すべての Java 開発ツールのなかでも最高です。
    • Java Enterprise Edition (EE) 6 のサポート: Java EE 6 仕様をサポートする最初の無料のオープンソース IDE です。
    • Java Standard Edition (SE) のサポート: 最新の Java SE 標準規格を使用してアプリケーションを開発できます。
  6. プロファイルおよびデバッグツール

    NetBeans IDE プロファイラによって、メモリー使用量および潜在的パフォーマンスボトルネックをリアルタイムで洞察することができます。さらに、プロファイル中のパフォーマンスの低下を避けるため、特定のコード部分だけを計測することができます。ヒープウォーカツールは、Java ヒープコンテンツの評価とメモリーリークの発見に役立ちます。

  7. 動的言語サポート

    NetBeans IDE は PHP、Groovy、JavaScript などのスクリプト言語の統合サポートを提供します。

    • PHP: NetBeans IDE for PHP を使用して、30mb 未満のダウンロードで、IDE の生産性 (コード補完、リアルタイムエラーチェック、デバッグなど) と、お気に入りのテキストエディタのスピードと簡単さの両方の世界の最高のメリットが得られます。
    • JavaScript: NetBeans IDE には、インテリジェントな JavaScript エディタ、CSS/HTML コード補完、Firefox および IE で JavaScript をデバッグする機能、バンドルされた人気のある JavaScript ライブラリなど、ユーザーが必要とする JavaScript ツールがあります。お気に入りの JavaScript フレームワークが全体の 80% を処理し、NetBeans IDE は残りの 20% で役立ちます。
    • Groovy: NetBeans IDE では、Grails アプリケーションを作成し、Groovy スクリプトを JavaSE project に組み込むことができるようになりました。
  8. 拡張可能なプラットフォーム

    その拡張可能なプラットフォームから始めて、独自の NetBeans IDE 機能および拡張機能を追加したり、IDE に似たアプリケーションを構築して、必要な機能だけを維持したりすることができます。プラットフォームとその Swing ベースの基盤の拡張により、開発期間を短縮し、パフォーマンスを最適化できます。

  9. カスタマイズ可能なプロジェクト

    NetBeans IDE 構築プロセスの過程において、専有の構築プロセスではなく、Apache Ant、make、Maven、rake などの業界標準規格に依存しているため、プロジェクトを簡単にカスタマイズし、機能を追加できます。IDE 以外のサーバーで、プロジェクトを構築、実行、配備できます。

  10. Java 以外のコードのサポート

    Java プログラミング言語に限定されません。C/C++ などの多くのほかのプログラミング言語、JavaScript などのスクリプト言語を含めることができます。さらに、独自の言語を定義して、それをプロジェクトに含めることもできます。

  11. 専用サポートの提供

    コミュニティーから必要な支援が得られない場合は、プログラミングのアドバイス、ソフトウェアのサポート、トレーニングの単位が提供される、Developer Support Packages を考慮してください。

次は何ですか。

  • 最新の NetBeans IDE をダウンロードしてください。
  • NetBeans IDE のガイド付きビデオをご覧になり、その強力な機能をいかに短時間で使えるようになるかを知ってください。
  • すべてのチュートリアルを探索してください。
  • 開発者とマスコミが NetBeans IDE についてどう言っているかをお読みください。
  • メーリングリストに参加するか、NetBeans ニュースレターを購読してください。
