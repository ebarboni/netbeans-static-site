NetBeans IDE へ切り替える方法


次の手順を追ったガイドに従って、ほかの IDE を NetBeans に切り替えることができます。

NetBeans への切り替え

NetBeans IDE の使用に関するチュートリアル、スクリーンキャスト、記事ほかの IDE から切り替えるには

Eclipse 開発者

既存の Eclipse プロジェクトの Java ソースファイルから NetBeans プロジェクトを作成できます (Eclipse プロジェクトメタデータがそのまま維持されます）。

「Eclipse プロジェクトの NetBeans IDE 6.5 へのインポート」をお読みください。

「NetBeans World Tour からの Eclipse インポート機能のデモ」をお読みください。

「NetBeansForEclipseUsers Wiki」をお読みください。

JBuilder 開発者

「JBuilder プロジェクトを NetBeans IDE にインポートする方法」をお読みください。(この記事は、以前のリリース用に書かれたものですが、NetBeans IDE 6.5 にもほとんど適用します。)

Java Studio 開発者

Java Studio Creator 開発者

「Java Studio Creator 2 Update 1 プロジェクトを NetBeans IDE へ移行する場合のヒント」(NetBeans 6.1 ドキュメントアーカイブを検索してください) をお読みください

Java Studio Enterprise 開発者

「Java Studio Enterprise 8.1 プロジェクトを NetBeans IDE へ移行する場合のヒント」(NetBeans 6.1 ドキュメントアーカイブを検索してください) をお読みください
