Razões principais para trocar para o NetBeans IDE


O ambiente de desenvolvimento integrado (IDE) do NetBeans entrega. O NetBeans IDE pode aumentar sua produtividade quando estiver trabalhando com o Java SE, Java EE, ou a tecnologia Java ME, assim como com o PHP, Groovy, JavaScript e C/C++. Ferramentas visuais que geram código esqueleto também estão disponíveis, permitindo que você crie aplicativos básicos sem escrever uma única linha de código.

Aqui estão as razões principais para usar o NetBeans IDE:
  1. Funciona fora da caixa

    Basta fazer o download e instale o NetBeans IDE e você está pronto para iniciar. A instalação é super fácil com seu pequeno tamanho de download. Todos as ferramentas e recursos do IDE estão totalmente integrados—não é preciso caçar plug-ins—e eles funcionam junto quando você inicia o IDE.

  2. Código-fonte aberto e grátis

    Quando você usa o NetBeans IDE, você participa de uma comunidade de código-fonte aberto vibrante, de milhares de usuários prontos para ajudar e contribuir. Há discussões nas listas de correio do NetBeans, blogs no PlanetNetBeans e FAQs que ajudam.

  3. Desenvolvedor conectado

    O NetBeans IDE é a ferramenta de escolha por equipes trabalhando em um ambiente colaborativo. Você pode criar e gerenciar projetos java.net hospedados, por exemplo; registrar relatórios de rastreamento de questões usando o Jira e o Bugzilla, e colaborar com os desenvolvedores—tudo diretamente da interface familiar do IDE.

  4. Poderoso GUI Builder

    O GUI Builder (anteriormente conhecido como Project Matisse) suporta um sofisticado e simplificadoSwing Application Framework e Beans Binding. Agora você pode construir IGUs de forma natural.

  5. Suporte para padrões e plataformas Java

    O IDE fornece soluções do princípio ao fim para todas as plataformas de desenvolvimento Java, incluindo os padrões Java mais recentes.

    • Java Mobility Support Ambiente completo para criar, testar e executar aplicativos para dispositivos móveis. Com blocos de pré-processador, você pode tratar imediatamente de questões de fragmentação. O suporte para o desenvolvimento Java Mobility é o melhor entre todas as ferramentas de desenvolvimento Java.
    • Suporte para o Java Enterprise Edition (EE) 6: O primeiro IDE grátis de código-fonte aberto para suportar as especificações do Java EE 6.
    • Suporte para o Java Standard Edition (SE): Você pode desenvolver aplicativos usando os padrões mais recentes do Java SE.
  6. Ferramentas de perfilamento e depuração

    Com o NetBeans IDE profiler, você pode obter um visão em tempo real do uso de memória e potenciais gargalos de desempenho. Mais ainda, você pode instrumentar partes específicas do código para evitar a degradação de desempenho durante o perfilamento. A ferramenta HeapWalker ajuda a avaliar o conteúdo do heap do Java e encontrar vazamentos de memória.

  7. Suporte para linguagem dinâmica

    O NetBeans IDE fornece o suporte integrado para linguagens de script, como o PHP, Groovy e o JavaScript.

    • PHP: Com o NetBeans IDE para o PHP, você obtém o melhor dos dois mundos: a produtividade de um IDE (conclusão de código, verificação de erro em tempo real, depuração e mais) com a velocidade e simplicidade de seu editor de texto favorito em um download de menos de 30mb.
    • JavaScript: O NetBeans IDE tem as ferramentas do JavaScript que você precisa: um editor JavaScript inteligente, conclusão de código CSS/HTML, a habilidade de depurar JavaScript no Firefox e no IE, e populares bibliotecas JavaScript incorporadas. Sua estrutura JavaScript favorita irá levá-lo a 80% do caminho, e o NetBeans IDE irá ajudá-lo com os 20% restantes.
    • Groovy: No NetBeans IDE, você pode criar aplicativos Grails, integrar scripts Groovy com seu projeto JavaSE.
  8. Plataforma extensível

    Comece com sua plataforma extensível e adicione seus próprios recursos e extensões NetBeans IDE ou construa um aplicativo to tipo IDE, mantendo somente os recursos que desejar. A extensão da plataforma e sua fundação com base no Swing economiza tempo de desenvolvimento e pode otimizar o desempenho

  9. Projetos personalizáveis

    Através do processo de compilação do NetBeans IDE, que se apóia em padrões da indústria como o Apache Ant, make, Maven e o rake, ao invés de um processo de compilação proprietário, você pode personalizar com facilidade projetos e adicionar funcionalidade. Você pode compilar, executar e implementar projetos em servidores fora do IDE.

  10. Suporte ao código não Java

    Você não está limitado à linguagem de programação Java. Você pode incluir muitas outras linguagens de programação, como o C/C++, linguagens de script como o JavaScript, etc. E ainda mais excitante, definir sua própria linguagem e incluí-la em seus projetos.

  11. Suporte dedicado disponível

    Quando não puder obter a ajuda necessária da comunidade, considere usar Pacotes de Suporte ao Desenvolvedor, que oferecem conselho de programação, suporte de software e créditos de treinamento.
