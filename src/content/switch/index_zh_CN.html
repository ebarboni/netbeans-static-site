切换到 NetBeans IDE

越来越多的开发者正在切换到 NetBeans IDE，以便利用提供的各种现成的功能。我们（NetBeans 团队）可以很容易告诉您为什么 NetBeans IDE 是一个非常棒的工具，但为什么不向您的同行打听打听呢？听听已进行切换的开发者讲述他们的故事。了解他们为什么要进行切换，NetBeans IDE 如何提高他们的工作效率以及您可以多么轻松地进行切换！


为什么要切换到 NetBeans IDE？

与其他 IDE 相比，NetBeans IDE 提供了一组丰富的功能和效率更高的工作环境。它可以直接使用：您不需要浪费宝贵的开发时间搜索插件。您可以通过 NetBeans IDE 创建最新技术的应用程序，无论它们是独立应用程序、Web 层应用程序、移动应用程序还是 J2EE 应用程序。

切换到 NetBeans

为了满足开发者从 Eclipse 和其他 IDE 轻松进行迁移的需求，NetBeans 提供了 Eclipse 项目导入模块，以便通过简单且易于实现的方式从基于 Eclipse 的项目过渡到 NetBeans IDE。

加入 NetBeans 社区

NetBeans 社区中每天都会发生很多精彩有趣的事情。为了帮助您了解最新的开发进展情况，您可以查看 NetBeans 路线图，订阅 NetBeans 新闻速递以及与其他开发者联系。您也可以加入 NetBeans 社区并做出自己的贡献：编写代码，构建插件，提交教程，在邮件列表上发布技术问题，等等。
