Главные причины для перехода на среду IDE NetBeans


Поставки интегрированной среды разработки NetBeans (integrated development environment, IDE). Среда IDE NetBeans может поднять вашу производительность при работе с технологиями Java SE, Java EE или Java ME, а также PHP, Groovy, JavaScript и C/C++. Визуальные средства, которые создают схему кода, также доступны, что позволяет создавать базовые приложения без написания кода.

Вот главные причины для использования среды IDE NetBeans:
  1. Сразу готова к работе

    Просто загрузите и установите среду IDE NetBeans — и можно начинать работу. Установка пройдет легко и быстро в силу маленького размера загружаемого файла. Все средства и функции среды IDE полностью интегрированы — нет необходимости разыскивать подходящие подключаемые модули — и все это взаимодействует, как только вы запускаете среду IDE.

  2. Бесплатно и с открытым исходным кодом

    Когда вы используете среду IDE NetBeans, вы присоединяетесь к живому сообществу открытого исходного кода, состоящему из тысячи пользователей, готовых помочь и поучаствовать в разработке. В сообществе вы найдете дискуссии по спискам рассылки NetBeans, блоги на PlanetNetBeans и полезные советы в разделе Часто задаваемые вопросы.

  3. Подключенный разработчик

    Среда IDE NetBeans — это средство, которое выбирают команды разработчиков для сотрудничества в среде для совместной работы над проектами. Например, вы можете создавать проекты, размещенные на java.net, и управлять ими; отчеты по отслеживанию ошибок в файлах при помощи Jira и Bugzilla и сотрудничество с единомышленниками — все это возможно через знакомый интерфейс среды IDE.

  4. Мощный GUI Builder

    GUI Builder (ранее известный как проект Matisse) поддерживает передовую и тем не менее очень простую Инфраструктуру приложений Swing и привязку компонентов. Теперь вы можете создавать графические интерфейсы пользователя естественным образом.

  5. Поддержка стандартов и платформ Java

    Среда IDE предоставляет сквозные решения для всех платформ разработки Java, включая последние стандарты Java.

    • Поддержка Java Mobility: полноценная среда для создания, тестирования и запуска приложений для мобильных устройств. При помощи блоков препроцессора можно быстро обработать ошибки фрагментации. Поддержка разработки Java Mobility здесь является лучшей среди всех средств разработки Java.
    • Поддержка Java Enterprise Edition (EE) 6: первая открытая среда IDE с открытым исходным кодом, поддерживающая спецификации Java EE 6.
    • Поддержка Java Standard Edition (SE): вы сможете разрабатывать приложения, соответствующие последним стандартам Java SE.
  6. Средства профилирования и отладки

    Средство профилирования среды IDE NetBeans позволяет вам проследить использование памяти и потенциальные факторы, снижающие производительность, в режиме реального времени. Более того, вы можете выделять средства для определенных частей кода, чтобы избежать потери производительности при профилировании. Средство HeapWalker поможет вам оценить содержимое "кучи" Java и найти потери доступа к памяти.

  7. Поддержка динамического языка

    Среда IDE NetBeans предоставляет встроенную поддержку для языков написания сценария, например PHP, Groovy и JavaScript.

    • PHP: работая в среде IDE NetBeans с PHP, вы получаете двойное преимущество: продуктивность среды IDE (автозавершение кода, проверка ошибок в режиме реального времени, отладка и пр.) и скорость и простоту вашего любимого текстового редактора, загрузив менее 30 Мб.
    • JavaScript: в среде IDE NetBeans есть все необходимые инструменты JavaScript: рациональный редактор JavaScript, автозавершение кода CSS/HTML, возможность деблокирования JavaScript в Firefox и IE, а также интегрированные популярные библиотеки JavaScript. Ваша любимая инфраструктура JavaScript — это 80% успеха, остальные 20% вам подарит среда IDE NetBeans.
    • Groovy: теперь в среде IDE NetBeans можно создавать приложения Grails, интегрировать сценарии Groovy в проекты JavaSE.
  8. Расширяемая платформа

    Начните с расширяемой платформы и добавьте собственные функции IDE NetBeans и расширения или создайте приложение, аналогичное среде IDE, оставив только необходимые функции. Расширяя платформу и его основы на базе Swing сэкономит время для разработки и улучшит показатели производительности.

  9. Настраиваемые проекты

    В процессе сборки среды IDE NetBeans, основывающемся скорее на отраслевых стандартах, например Apache Ant, make, Maven и rake, чем на частных процессах сборки, можно с легкостью настроить проекты и добавить функции. Вы сможете собирать, запускать и развертывать проекты на серверах вне среды IDE.

  10. Поддержка кода, отличного от Java

    Вам не обязательно ограничиваться языком программирования Java. В проект можно включать многие другие языки программирования, например C/C++, языки написания сценария, например JavaScript, и т.д. Можно также определить собственный язык и включить его в проект.

  11. Специальная поддержка

    Если вы не можете получить поддержку сообщества, обратите внимание на Пакеты поддержки разработчика, в которых вы найдете советы по программированию, поддержку ПО и обучающие материалы.
