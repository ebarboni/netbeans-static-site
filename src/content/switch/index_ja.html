NetBeans IDE への切り替え

NetBeans IDE に切り替えて、すべてがすぐに使える機能を利用している開発者が増えています。NetBeans チームが、なぜ NetBeans IDE が優れたツールであるのかをお伝えすることもできますが、同業者のお話を聞かれるのはいかがでしょうか。切り替えを行なった開発者のお話をお読みください。なぜ切り替えたのか、NetBeans IDE はどう生産性を向上するか、いかに簡単に切り換えることができるかを知ってください。


なぜ NetBeans IDE へ切り替えるのか

NetBeans IDE は豊富な機能セットを備え、ほかの IDE よりも生産性の高い作業環境を提供します。さらに、そのままですぐに機能します。貴重な開発の時間をプラグインの検索に費やすこともありません。NetBeans IDE を使用することで、スタンドアロンでも、Web 層でも、モバイルでも、J2EE でも最先端のアプリケーションを作成できます。

「NetBeans IDE に切り替える主な理由」をお読みください。

NetBeans へ切り替える

Eclipse やほかの IDE からの簡単な移行に対する開発者の要望に直接こたえて、NetBeans では、Eclipse ベースのプロジェクトの NetBeans IDE への簡単で実装しやすい移行のための Eclipse プロジェクトインポートモジュールを提供しています。

切り替える方法で詳細をお読みください。

NetBeans コミュニティーに参加する

NetBeans コミュニティーではたくさんのすてきなことが起こっています。最新の開発を完全に把握するために、NetBeans ロードマップをチェックし、NetBeans ニュースレターを購読して、ほかの開発者とつながりを持つことができます。また、コードを書く、プラグインを構築する、チュートリアルを提出する、メーリングリストで技術的質問に対応するなどで、NetBeans コミュニティーに貢献することによって、活動に参加することもできます。
 
NetBeans コミュニティーページで詳細を確認してください。
