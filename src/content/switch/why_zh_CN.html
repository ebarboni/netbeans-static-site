切换到 NetBeans IDE 的主要原因


提供了 NetBeans 集成开发环境 (IDE)。在处理 Java SE、Java EE 或 Java ME 技术以及 PHP、Groovy、JavaScript 和 C/C++ 时，NetBeans IDE 可以显著提高您的工作效率。还可以使用生成框架代码的可视化工具，以便创建基本应用程序，而无需编写一行代码。

下面是使用 NetBeans IDE 的主要原因：
  1. 直接使用

    只需下载并安装 NetBeans IDE，您就可以开始工作了。下载文件很小，安装起来轻而易举。所有 IDE 工具和功能完全集成在一起（不需要查找插件），在启动 IDE 时，它们将一起使用。

  2. 免费且开源

    在使用 NetBeans IDE 时，您将加入一个充满活力的开源社区，该社区拥有数千个用户，他们可以随时为您提供帮助和做出贡献。您可以查看 NetBeans 邮件列表中提供的讨论内容、PlanetNetBeans 上发布的博客以及非常有用的常见问题解答。

  3. 团队开发

    NetBeans IDE 是在协作环境中工作的团队的首选工具。您可以创建和管理 java.net 托管的项目（例如，使用 Jira 和 Bugzilla 的文件问题跟踪报告），以及与志同道合的开发者进行合作，所有操作都是直接在熟悉的 IDE 界面中完成的。

  4. 强大的 GUI 生成器

    GUI 生成器（以前称为 Project Matisse）支持先进但简化的 Swing 应用程序框架和 Beans 绑定。现在，您可以使用通常方法构建 GUI。

  5. Java 标准和平台支持

    IDE 为所有 Java 开发平台提供了端对端的解决方案，其中包括最新的 Java 标准。

    • Java Mobility 支持：用于为移动设备创建、测试和运行应用程序的完善环境。通过使用预处理器块，您可以轻松处理分段问题。Java Mobility 开发支持是所有 Java 开发工具中最好的。
    • Java Enterprise Edition (EE) 6 支持：第一个支持 Java EE 6 规范的免费开源 IDE。
    • Java Standard Edition (SE) 支持：您可以使用最新的 Java SE 标准开发应用程序。
  6. 性能分析和调试工具

    通过使用 NetBeans IDE Profiler，您可以实时查看内存使用情况和潜在的性能瓶颈。此外，您还可以选择分析特定的代码部分，以免在分析性能期间出现性能下降。HeapWalker 工具可帮助您评估 Java 堆内容和查找内存泄漏问题。

  7. 动态语言支持

    NetBeans IDE 为脚本语言提供集成支持，例如，PHP、Groovy 和 JavaScript。

    • PHP：通过使用 NetBeans IDE 进行 PHP 开发，您可以获得两方面的好处：提高 IDE 的工作效率（代码完成、实时错误检查和调试等）并使用快捷简单的常用文本编辑器，下载文件小于 30 MB。
    • JavaScript：NetBeans IDE 具有您所需的 JavaScript 工具：智能 JavaScript 编辑器、CSS/HTML 代码完成、在 Firefox 和 IE 中调试 JavaScript 的功能以及绑定的常用 JavaScript 库。您喜爱的 JavaScript 框架可以完成您的 80% 工作，而 NetBeans IDE 帮助您完成剩余的 20% 工作。
    • Groovy：在 NetBeans IDE 中，您现在可以创建 Grails 应用程序，将 Groovy 脚本与您的 JavaSE 项目集成在一起。
  8. 可扩展的平台

    可以从它的可扩展平台入手，并添加您自己的 NetBeans IDE 功能和扩展或构建类似 IDE 的应用程序，从而仅保留所需的功能。通过扩展平台及其基于 Swing 的基础，可以节省开发时间和优化性能。

  9. 可定制的项目

    通过 NetBeans IDE 构建过程（取决于行业标准，如 Apache Ant、make、Maven 和 rake），而不是专用构建过程，您可以方便地定制项目和添加功能。您可以在 IDE 外部构建和运行项目以及将项目部署到服务器中。

  10. 非 Java 代码支持

    您并不仅限于使用 Java 编程语言。您可以包含很多其他编程语言，例如，C/C++ 和脚本语言（如 JavaScript）等。更令人振奋的是，您可以定义自己的语言，并在您的项目中包含该语言。

  11. 提供了专用支持

    如果您无法从社区中获得所需的帮助，请考虑使用开发者支持软件包，其中提供了编程建议、软件支持以及培训学分。
