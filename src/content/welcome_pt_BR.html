Bem-vindo ao Projeto NetBeans!

1. O que é o Projeto NetBeans?

O NetBeans IDE é um Ambiente de Desenvolvimento Integrado de código-fonte aberto, uma ferramenta de software que você pode baixar e usar de graça para desenvolver aplicativos. Mais...

A Plataforma NetBeans é uma estrutura extensível que pode ser usada para desenvolver aplicativos desktop profissionais independentes do SO. Mais...

Finalmente, o NetBeans.org é uma comunidade crescente de código-fonte aberto com centenas de desenvolvedores experientes localizados ao redor do mundo. Mais...

2. Como posso iniciar?

Faça o download do NetBeans IDE. Ele tem todo o necessário para desenvolver aplicativos de classe mundial. O software NetBeans é 100% baseado no IDE e plataforma Java portanto, funciona em cada SO onde o JDK estiver disponível. O NetBeans suporta uma ampla gama de tecnologias Java, mas não está limitado ao Java -outras linguagens também são suportadas. A ferramentas do NetBeans tem tanta funcionalidade, que você pode baixar aquelas que forem melhor para você. Aprenda mais sobre as tecnologias suportadas e sobre os recursos na página Recursos. Você pode adicionar mais funcionalidades com módulos plug-in (no IDE, escolha Ferramentas > Plug-ins).

3. Como posso me envolver?

Há uma crescente comunidade de usuários do NetBeans no mundo, variando de novos usuários aos gurus. Junte-se a eles para discutir dicas e truques, recursos do NetBeans, solução de problemas, e muito mais. Registre-se agora!
