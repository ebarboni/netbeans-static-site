Индивидуальная настройка шаблона Java NetBeans в соответствии с потребностями

Составитель - Артур Ченг (Arthur Cheng), в рамках конкурса Выигрывайте с NetBeans

В NetBeans предусмотрен набор шаблонов общих классов. Однако зачастую пользователям приходится повторно вводить или вырезать и вставлять один и тот же текст каждый раз, когда требуется создать новый класс (например, текст об авторских правах, сообщение об ограничении ответственности и т.д.).

Вместо этого в NetBeans можно создать класс Java, уже содержащий эту информацию. Это довольно просто. Достаточно воспользоваться настроенным шаблоном.

Предположим, требуется создать индивидуально настроенный шаблон класса Java, содержащий информацию о компании. Далее перечислены шаги для настройки такого шаблона (они также применяются для индивидуальной настройки интерфейсов Java, апплетов и т.д.).

  1. Выберите команду "Средства" > "Параметры". В окне "Параметры" выберите "Создание и управление исходным кодом" > "Шаблоны" > "Классы Java" > "Класс Java".
  2. Щелкните правой кнопкой мыши элемент "Класс Java" и выберите "Копировать".
  3. Вернитесь по пути "Создание и управление исходным кодом" > "Шаблоны" > "Классы Java".
  4. Щелкните правой кнопкой мыши элемент "Классы Java" и выберите "Вставить > Копировать".
  5. Среди шаблонов в списке "Классы Java" должен отображаться шаблон Class_1.java.
  6. Щелкните правой кнопкой мыши Class_1.java и выберите "Переименовать". Откроется следующее окно. Измените имя шаблона на имя компании (в данном примере - XYZCompany) и нажмите "ОК". (Имя шаблона не должно завершаться словом "Template". В этом случае новый шаблон не будет отображаться в NetBeans).

    В следующем всплывающем окне нажмите кнопку "Далее".

    Затем появится новый шаблон класса Java.

  7. Шаблон Class_1.java переименован в XYZCompany.java. Это лишь первая часть процедуры.
  8. Щелкните шаблон XYZCompany.java правой кнопкой мыши и выберите "Открыть". В редакторе исходного кода откроется исходный код шаблона. В окне "Параметры" выберите "Закрыть" и перейдите к редактору исходного кода.
  9. В редакторе исходного кода соответствующим образом измените исходный код. Обратитесь к приведенному ниже примеру (НЕ изменяйте объявление класса. Все прочие имена классов можно заменить макросом __ИМЯ__, который замещается именем актуального класса при создании файла на основе шаблона) :
    /*
 * __ИМЯ__.java
 *
 * Создано __ДАТА__, __ВРЕМЯ__
 *
 * Copyright(c) {ГОД!!!} XYZ Company, Inc.  Все права защищены.
 * Это программное обеспечение является собственностью XYZ Company.
 *
 */

  10. После завершения выберите "Файл" > "Сохранить".
  11. Теперь при необходимости создания класса Java Достаточно выбрать создание нового класса Java XYZCompany, и вся указанная информация будет добавлена автоматически.

Настройка имени пользователя по умолчанию и других макросов

  1. Выберите команду "Средства" > "Параметры". В окне "Параметры" выберите "Редактирование" > "Исходные коды Java".
  2. Нажмите кнопку с многоточием (...), соответствующую значению "Таблица строк" из вкладки "Свойства", расположенной справа.
  3. В поле "ПОЛЬЗОВАТЕЛЬ" укажите свое имя.
  4. Добавьте новый макрос в форме {Имя макроса}={Значение макроса} и нажмите "ОК". (При определении макроса не следует заменять символы подчеркивания в начале и в конце имени макроса.)
