定制 NetBeans Java 模板以满足您的需求

撰稿人：Arthur Cheng，与 NetBeans 共赢参赛作品

NetBeans 提供了一些通用类模板。不过，很多时候，您可能会发现每次尝试创建新的类时，您重复键入或剪切并粘贴相同的通用文本（如贵公司的版权文本、免责声明等）。

让 NetBeans 创建一个包含该信息的 Java 类岂不是更好？确实如此，可以很容易做到这一点。您可以使用定制模板来解决该问题！

比如说，您希望创建一个定制 Java 类模板以包含贵公司的信息。下面是设置该模板的步骤（这些步骤也适用于定制 Java 接口、Applet 等），让我们开始创建吧。

  1. 选择 "Tools"（工具）> "Options"（选项）。在 "Options"（选项）窗口中，浏览到 "Source Creation and Management"（源代码创建和管理）> "Templates"（模板）> "Java Classes"（Java 类）> "Java Class"（Java 类）。
  2. 右键单击 "Java Class"（Java 类），然后选择 "Copy"（复制）。
  3. 返回到 "Source Creation and Management"（源代码创建和管理）> "Templates"（模板）> "Java Classes"（Java 类）。
  4. 右键单击 "Java Classes"（Java 类），然后选择 "Paste"（粘贴）> "Copy"（复制）。
  5. 现在，您应该看到 Class_1.java，它是 "Java Classes"（Java 类）下面的模板之一。
  6. 右键单击 Class_1.java，然后选择 "Rename"（重命名），将弹出以下窗口。将该名称修改为贵公司的名称（我们在此处使用 XYZCompany），然后单击 "OK"（确定）。（模板名称不要以文字 "Template" 结尾。如果按这种方式命名新模板，NetBeans 将不显示该模板。）

    在以下弹出窗口中，单击 "Next"（下一步）。

    瞧，新的 Java 类模板显示出来了。

  7. 现在，您可以看到 Class_1.java 重命名为 XYZCompany.java。此时，我们的工作只完成了一半。
  8. 右键单击 XYZCompany.java，然后选择 "Open"（打开）。将在源代码编辑器中打开模板源代码。在 "Options"（选项）窗口中单击 "Close"（关闭）以将其关闭，然后转到源代码编辑器。
  9. 最神奇的地方就要开始了 ... 在源代码编辑器中，您可以根据需要修改源代码。请参见下面的示例（不要更改类声明。您可以将所其他类名替换为 __NAME__ 宏，在通过模板创建文件时，将替换为实际类名）：
    /*
 * __NAME__.java
 *
 * Created on __DATE__, __TIME__
 *
 * Copyright(c) {YEAR!!!} XYZ Company, Inc.  All Rights Reserved.
 * This software is the proprietary information of XYZ Company.
 *
 */

package Templates.Classes;

                                //  to most of your classes

    private static final Logger log = Logger.getLogger (__NAME__.class);
    private static final String CLASS_NAME = "__NAME__";
    
    
    
    /**
     * Creates a new instance of __NAME__
     */
    public XYZCompany ()

    /**

}    //  end __NAME__
  10. 完成后，单击 "File"（文件）> "Save"（保存）。
  11. 现在，每次您需要创建 Java 类时，您只需选择创建新的 XYZCompany Java 类，其中将包含所有的定制信息！

设置缺省用户名和其他宏

  1. 选择 "Tools"（工具）> "Options"（选项）。在 "Options"（选项）窗口中，展开 "Editing"（编辑）> "Java Sources"（Java 源代码）。
  2. 从右侧的“属性”标签中，单击 "Strings Table"（字符串表）值的省略号 (...) 按钮。
  3. 设置 USER=您的名称。
  4. 在 {宏名称}={宏值} 表单中添加其他宏，然后单击 "OK"（确定）。（在定义宏时，不要将下划线字符放在宏名称的开头和结尾处。）
