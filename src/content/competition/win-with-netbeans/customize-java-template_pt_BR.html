Personalize seu modelo Java do NetBeans para se ajustar às suas necessidades

Contribuição de Arthur Cheng como parte da competição Win With NetBeans

O NetBeans vem com modelos de classe genérica. No entanto, com frequência, você tem de redigitar ou cortar e colar o memso texto genérico cada vez que tenta criar uma nova classe (tal como o texto de direito autoral de sua empresa, renuncia, etc.)

Não seria bom que o NetBeans criasse a classe Java que já contém a informação? Sim, isso pode ser feito com facilidade. Modelo personalizado para o resgate!

Vamos dizer que você deseja criar um modelo Java personalizado com as informações de sua empresa. Aqui estão as etapas para configurá-lo (e estas etapas se aplicam também para personalizar interfaces Java, applets, etc.), portanto, aqui vamos nós.

  1. Escolha Ferramentas > Opções. Na janela Opções, navegue para Criação e gerenciamento de código-fonte > Modelos > Classes Java > Classe Java
  2. Clique com o botão direito do mouse em Classe Java e escolha Copiar
  3. Volte para Criação e gerenciamento de código-fonte > Modelos > Classes Java
  4. Clique com o botão direito do mouse em Classes Java e escolha Colar > Copiar
  5. Você agora deveria ver Class_1.java como um dos modelos sob Classes Java.
  6. Clique com o botão direito do mouse em Class_1.java e escolha Renomear e a seguinte janale aparece. Substitua o nome pelo nome de sua empresa (nós iremos usar XYZCompany aqui) e clique em OK. (Não termine o nome do modelo com a palavra Modelo. O NetBeans não irá exibir seu novo modelo caso você o nomeie desta forma).

    Clique em Avançar na janela suspensa seguinte.

    e pronto! seu novo modelo de classe Java aparece

  7. Você agora deveria ver a Class_1.java renomeada como XYZCompany.java. Chegamos a metade do caminho.
  8. Clique com o botão direito do mouse em XYZCompany.java e escolha Abrir. O modelo do código-fonte é aberto no Editor de código-fonte. Clique em Fechar na janela Opções para fechâ-lo e ir para o Editor de código-fonte.
  9. Isso é onde a mágica começa... No Editor de código-fonte, modifique o código-fonte da forma que lhe aprouver. Veja o exemplo abaixo (NÃO altere a declaração da classe.   10. Quando tiver terminado, clique em Arquivo > Salvar.
  11. Agora, sempre que você precisar criar uma classe Java, basta escolher criar uma nova classe Java XYZCompany e terá a informação personalizada direito aqui!

Definir o nome de usuário padrão e outras macros

  1. Escolha Ferramentas > Opções. Na janela Opções, expanda Editando > Código-fonte Java
  2. Clique no botão de elipse (...) do valor da Tabela de strings na guia Propriedades na direita.
  3. Set USER=Your Name
  4. Adicione outra macros no formato {Macro Name}={Macro Value} e clique em OK. (Quando você define a macro, não coloque os caracteres de sublinhado no início e no fim do nome da macro).
