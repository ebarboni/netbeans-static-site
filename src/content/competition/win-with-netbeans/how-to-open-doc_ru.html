Полезный совет: Открытие любых типов документов в NetBeans

Составитель - Венсан Брабан (Vincent Brabant), в рамках конкурса Выигрывайте с NetBeans

Как в NetBeans можно открыть документ .doc с помощью MS Word, документ .XLS с помощью Excel, документ .PPT с помощью Powerpoint, ...
Каким образом в NetBeans можно открыть соответствующее приложение при щелчке на файле .LNK (часто отображаются на рабочем столе).

На самом деле это не так сложно.

Как известно, в NetBeans можно открывать файлы PDF.

При первом щелчке на файле PDF открывается следующее диалоговое окно:



Необходимо выбрать исполняемый файл. Логичным решением будет выбор Acrobat Reader или любого другого средства просмотра PDF, доступного на рынке.
Однако на самом деле можно выбрать абсолютно любой исполняемый файл.

В этом и заключается суть рекомендации.

В Windows 95, 98 и ME в каталоге команд существует исполняемый файл Start.exe.
Он позволяет получать файл как аргумент. Это может быть любой тип файла. Он выполняет поиск приложения, соответствующего расширению файла, и запускает его.


Таким образом, можно сделать следующее.

Нажмите кнопку "Выбрать" и перейдите к каталогу Windows, а затем выберите каталог команд и выполните поиск файла Start.exe.



Выделите его и нажмите кнопку "Выбрать". Теперь выбранным средством просмотра будет не acroread, а абсолютное имя Start.exe.



После этого Acrobat Reader запускается без проблем и позволяет просматривать выбранный файл PDF.
Как это возможно? Start.exe получает имя файла PDF в качестве параметра и выполняет поиск приложения, соответствующего такому расширению.
Поскольку на данном ПК формат PDF привязан к Acrobat Reader, открывается именно это приложение.

Проблема состоит в том, что исполняемый файл Start.exe доступен только в ОС Windows 95, 98 и ME. В Windows NT, 2000 и XP он отсутствует.
Однако в этих системах по-прежнему существует команда "Запуск". Она реализована как расширение командного процессора.
Для решения проблемы необходимо выполнить следующее:
1. Создайте файл .BAT в каталоге C:\.
Этому файлу BAT будет присвоено имя nbLauncher.bat.
Файл BAT содержит следующее:
start %1
2. Выберите файл BAT в качестве текущего средства просмотра.



Как теперь можно отрывать другие файлы, например .DOC, .XLS, .EXE или .LNK?

В главном меню выберите "Средства" | "Параметры".
Откроется панель "Параметры".
Выберите "Настройка среды IDE" | "Система" | "Типы объектов" и выберите "Документы PDF".
На панели "Свойства" выберите "Расширения и типы MIME...".


По умолчанию в списке элементов отображает только PDF, но для запуска из NetBeans можно добавить любые другие расширения.
Например, в данном примере добавлены расширения doc, xls, lnk и exe.


После завершения нажмите кнопку "ОК".

Затем выберите "Закрыть".

Теперь необходимо добавить в "Избранное" каталог, содержащий документы или исполняемые файлы. Также в "Избранное" можно добавить рабочий стол.
При щелчке по любому файлу .LNK запустится соответствующее приложение.
При щелчке по файлу .DOC документ откроется в программе MS Word.
