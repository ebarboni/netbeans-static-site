Согласно концепции открытого исходного кода, вопросы для этой викторины были предоставлены членами сообщества NetBeans. Все участники, правильно ответившие на вопрос, примут участие в розыгрыше четырех руководств NetBeans Field Guide (издание 2). Один человек может принять участие только один раз.

Благодарим всех, кто прислал свои вопросы! Некоторые присланные вопросы были очень похожи, поэтому мы случайным образом выберем пять авторов, которые получат футболку NetBeans 5.5.

Вопрос 8

Как открыть в среде IDE файлы, не входящие в проект NetBeans (например файлы HTML или TXT)?

Это невозможно, среда IDE предназначена только для исходного кода

Нужно использовать подключаемый модуль Importer для их импорта

Добавить их в окно "Избранное" с помощью контекстного меню

Открыть их в окне "Файлы" с помощью меню "Открыть"
