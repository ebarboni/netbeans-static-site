I live in [country], can I play?

This contest is open only to the legal residents of India who have reached the age of majority at the time of entry. See the contest rules for details.

I already have NetBeans IDE, how do I get the quiz plugin?

You install the quiz plugin directly from the NetBeans IDE. Go to Tools -> Plugins and find the quiz plugin in the list of available plugins. Select it for installation and click the Install button.

I'm having trouble connecting to the NetBeans Update Center. Can you help?

Check your proxy settings. Go to Tools -> Options -> General. Specify the general proxy settings the IDE uses to access the Internet by choosing one of the following: No Proxy, Use System Proxy Settings, or Manual Proxy Settings. To use Manual Proxy settings, you must specify the HTTP Proxy and Proxy Port. The HTTP Proxy is the host name or numeric IP address of the proxy server to use to connect to HTTP and FTP sites. The Proxy Port is the port number of the proxy server. Click More to configure more advanced proxy settings.

What versions of the NetBeans IDE can I use for the quiz?

NetBeans IDE 5.5 and later: NetBeans IDE 5.5, 5.5.1, 6.0, 6.0.1, 6.1 Beta and 6.1 Release Candidate.

I'm upgrading my Operating System during the quiz. Will I lose all my previous quizzes?

No, you can accumulate quiz points even if you install a new quiz plugin. We keep track of you by your email address. Use the same email address throughout the quiz.

Three of us are sharing the same computer, how can I set it up so we all can take quiz?

Set a different userdir for each person taking the quiz. Bu default, userdir is a hidden directory called .netbeans stored in the user's home directory under .netbeans/[nb-version]. The home directory is ${HOME} on Unix-like systems, C:\Documents and Settings\[user] on Windows. For more information, read the What is the .netbeans directory? and Can I run NetBeans with an alternate "userdir"? FAQs.

How are the winners selected for the weekly prize?

When the quiz is over, we generate a list of all the entrants who sent correct answers for all questions and then randomly choose one entrant. See the algorithm in pseudo-code: 

      
 if (now == Algorithm.End_Date) {

   Array correct= find(users who answered correctly all questions in quiz)

   int winnerIdx = Random() // randomly chose index

   return winner = correct[winnerIndex]

 }

As implementation is used: java.util.Random 

      
int winnerIdx = new Random().nextInt(correct.size());

How do I know if I've won a weekly quiz?

All winners will be notified by e-mail and list of winners will be posted weekly on the Quiz Winners page.

How do I win the grand prize?

The grand prize will be awarded to the person submits the most correct answers cumulatively from the weekly quizzes. In the event of a tie, the grand prize winner will be determined by random drawing from the top entrants.

How many quizzes are there?

Eleven. Quizzes are weekly starting April 14, 2008.

What's the deadline for submitting quizzes?

Weekly quizzes start Monday at 12:01 PM IST and end the following Monday at 11:59 PM IST.

I just found out about this contest and it started weeks ago. Can I take quizzes from previous weeks?

Yes! The Quiz plugin will tell you about quizzes from previous weeks.

Why should I bother taking old quizzes if someone else has won the weekly prize?

Because the grand prize is based on your cummulative score on all quizzes.
