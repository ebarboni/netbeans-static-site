Eu vivo no [país], posso participar?

Esse concurso somente está aberto para os residentes legais da Índia que atingiram a maioridade na data de sua entrada. Consulte as regras do concurso para obter detalhes.

Já tenho o NetBeans IDE, como obtenho o plug-in do quiz?

Você instala o plug-in do quiz diretamente do NetBeans IDE. Siga para Ferramentas -> Plug-ins e encontre o plug-in do quiz na lista de plug-ins disponíveis. Selecione-o para instalação e clique no botão Instalar.

Estou tendo dificulades para me conectar ao Centro de Atualização do NetBeans. Podem ajudar?

Verifique suas configurações do proxy. Siga para Ferramentas -> Opções -> Geral. Especifique as configurações gerias do proxy que o IDE usa para acessar a Internet ao escolher ums das seguintes: nenhum proxy, Usar as configurações do proxy do sistema ou Configurações manuais do proxy. Para usar as Configuração manuais do proxy, é preciso especificar o HTTP do proxy e a Porta do proxy. O HTTP do proxy é o nome do host ou endereço IP numérico que o servidor proxy usa para se conectar aos sites HTTP e FTP. A porta do proxy é o número da porta do servidor proxy. Clique em Mais para definir configurações mais avançadas do proxy.

Quais versões do NetBeans IDE posso usar para o quiz?

NetBeans IDE 5.5 e posterior: NetBeans IDE 5.5, 5.5.1, 6.0, 6.0.1, 6.1 Beta e 6.1 Release Candidate.

Estou atualizando meu Sistema Operacional durante o quiz. Irei perder todos meus quizzes anteriores?

Não, você pode acumular pontos de quiz mesmo se for instalar um novo plug-in de quiz. Nós o acompanhamos por seu endereço de e-mai. Use o mesmo endereço de e-mail durante o quiz.

Três de nós estamos compartilhando o mesmo computador, como posso configurá-lo para que todos nós possamos participar do quiz?

Defina uma userdir diferente para cada pessoa que vá participar do quiz. Por padrão, userdir é um diretório oculto denominado netbeans armazenado no diretório principal do usário sob .netbeans/{nb=version]. O diretório principal é ${HOME} em sistema do tipo Unix e C:\Documents and Settings\[user] no Windows. Para obter mais informações consulte as FAQS O que é o diretório .netbeans? e Posso executar o NetBeans com um "userdir" alternativo?.

Como são selecionados os ganhadores para o prêmio semanal?

Quando o quiz termina, nós geramos uma lista dos participantes que enviaram as respostas corretas para todas as perguntas e então escolhemos aleatoriamente um participante. Consulte o algorítmo em pseudo-código: 

      
 if (now == Algorithm.End_Date) {

   Array correct= find(users who answered correctly all questions in quiz)

   int winnerIdx = Random() // randomly chose index

   return winner = correct[winnerIndex]

 }

Como implementação usamos: java.util.Random 

      
int winnerIdx = new Random().nextInt(correct.size());

Com posso saber se ganhei um quiz semanal?

Todos os ganhadores serão notificados por e-mail e uma lista dos ganhadores será semanalmente publicada napágina Ganhadores do Quiz.

Como posso gannhar o grande prêmio?

O grande prêmio será concedido à pessoas que submente as respostas mais correta cumulativamente dos quizzes semanais. Caso haja empate, o ganhador do grande prêmio será determinado de forma aleatória entre os participantes do topo.

Qual é o prêmio da próxima semana?

Os prêmios estão listados na página Prêmios

Onde posso ver a lista dos ganhadores?

A lista de ganhadores será publicada semanalmente na página Ganhadores do Quiz.

Quantos quizzes há?

Onze. Os quizzes são semanais iniciando em 14 de abril de 2008.

Qual o prazo para submeter quizzes?

Os quizzes semanais iniciam na segunda-feira as 12:00 IST e terminam na próxima segunda-feira as 23:59 IST.

Acabei de saber sobre este concurso e ele começou semanas atrás. Posso participar de quizzes de semanas anteriores?

Sim! O plug-in do quiz irá informá-lo sobre os quizzes de semanas anteriores.

Porque devo me incomodar em participar de quizzes antigos e outra pessoa ganhou o prêmio?

Porque o grande prêmio tem base em sua pontuação cumulativa em todos os quizzes.
