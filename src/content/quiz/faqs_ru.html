Я живу в [название страны], могу ли я участвовать?

Это состязание открыто только для лиц, законно проживающих в Индии, достигших совершеннолетия на момент участия. Дополнительные сведения приведены в правилах состязания

У меня уже установлена среда IDE NetBeans, как мне получить подключаемый модуль викторины?

Установка подключаемого модуля выполняется непосредственно из среды IDE NetBeans. Откройте пункт меню "Сервис -> Подключаемые модули" и выберите подключаемый модуль викторины в списке доступных подключаемых модулей. Выберите его для установки и нажмите кнопку "Установить".

У меня возникли затруднения при подключении к центру обновлений NetBeans. Можете ли вы помочь?

Проверьте параметры настройки прокси-сервера. Выберите пункт меню "Сервис -> Параметры -> Общие". Укажите общие параметры прокси-сервера, используемые в среде IDE для доступа к Интернету путем выбора одного из следующих вариантов: "Без прокси", "Использовать системные параметры прокси", или "Ручная настройка прокси". В случае выбора варианта "Ручная настройка прокси" необходимо указать прокси-сервер HTTP и порт прокси. Прокси-сервер HTTP — это имя узла или IP-адрес прокси-сервера, используемого при подключении к узлам HTTP и FTP. Порт прокси — это номер порта на прокси-сервере. Нажмите кнопку Дополнительно для настройки расширенного набора параметров прокси-сервера.

Какие версии среды IDE NetBeans я могу использовать для викторины?

Среду IDE NetBeans 5.5 и более поздних версий: среду IDE NetBeans 5.5, 5.5.1, 6.0, 6.0.1, бета-версии 6.1 и 6.1 Release Candidate.

Я обновляю свою операционную систему во время викторины. Будут ли все мои предыдущие викторины утрачены?

Нет очки викторины можно накапливать даже в случае установки подключаемого модуля викторины заново. Мы записываем очки на адрес электронной почты. В течение всей викторины следует использовать один и тот же адрес электронной почты.

Нас трое на одном компьютере, как его настроить, чтобы мы все могли принять участие?

Установите разные каталоги пользователя для каждого участника викторины. По умолчанию каталог пользователя — это скрытый каталог с именем .netbeans, который хранится в домашнем каталоге пользователя по пути .netbeans/[nb-version]. Домашний каталог — ${HOME} на Unix-подобных системах, и C:\Documents and Settings\[user] в Windows. Более подробные сведения приведены в ответах на часто задаваемые вопросы Что такое каталог .netbeans? и Можно ли запустить NetBeans с другим каталогом "userdir"?.

Как выбираются победители, выигравшие еженедельный приз?

По завершении викторины мы формируем список всех участников, отправивших правильные ответы на все вопросы, а затем случайным образом выбираем одного участника. Используется следующий алгоритм (приведен псевдокод): 

      
 if (now == Algorithm.End_Date) {

   Array correct= find(правильно ответившие на все вопросы викторины пользователи)

   int winnerIdx = Random() // выбирается случайный индекс

   return winner = correct[winnerIndex]

 }

Реализация случайной выборки: 

      
int winnerIdx = new Random().nextInt(correct.size());

Как я узнаю о выигрыше в еженедельной викторине?

Всем победителям отправляется уведомление по электронной почте, и список победителей еженедельно размещается на странице победителей викторины.

Как выиграть гран-при?

Гран-при присуждается участнику, отправившему максимальное суммарное количество правильных ответов в еженедельных викторинах. В случае наличия нескольких таких участников, гран-при определяется жеребьевкой среди них.

Какой будет приз на следующей неделе?

Список призов приведен на странице призов

Где список победителей?

Список победителей еженедельно размещается на странице победителей викторины.

Сколько всего викторин?

Одиннадцать. Викторины проводятся еженедельно, начиная с 14 апреля 2008 года.

Каков крайний срок отправки викторин?

Еженедельные викторины начинаются в понедельник в 12:01 PM IST и заканчиваются в следующий понедельник в 11:59 PM IST.

Я только что узнал об этом состязании, а оно началось несколько недель назад. Могу ли я участвовать в викторинах за прошлые недели?

Да! Подключаемый модуль викторин выводит сведения о викторинах за предыдущие недели.

Зачем участвовать в старых викторинах, если еженедельный приз уже выигран?

Гран-при присуждается по совокупному счету во всех викторинах.
