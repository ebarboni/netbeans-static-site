我居住在[国家/地区]，我可以参加测验吗？

本次比赛仅向参加时已达到成人年龄的印度合法公民开放。有关详细信息，请参见比赛规则。

我已安装了 NetBeans IDE，我如何获取测验插件？

您可以直接从 NetBeans IDE 中安装测验插件。转至“工具”->“插件”，然后在可用插件列表中找到测验插件。选择该插件，然后单击“安装”按钮以进行安装。

我在连接到 NetBeans 更新中心时遇到问题。您能帮帮忙吗？

检查您的代理设置。转至“工具”->“选项”->“常规”。选择以下选项之一以指定 IDE 访问 Internet 时使用的常规代理设置：“无代理”、“使用系统代理设置”或“手动代理设置”。要使用手动代理设置，您必须指定 HTTP 代理和代理端口。HTTP 代理是指代理服务器用于连接到 HTTP 和 FTP 站点的主机名或数字 IP 地址。代理端口是指代理服务器的端口号。单击“更多”以配置其他高级代理设置。

我可以使用哪种版本的 NetBeans IDE 参加测验？

NetBeans IDE 5.5 和更高版本：NetBeans IDE 5.5、5.5.1、6.0、6.0.1、6.1 Beta 和 6.1 Release Candidate。

在测验期间，我正在升级操作系统。我会丢失以前的所有测验吗？

不会，即使您安装了新的测验插件，您仍然可以积累测验积分。我们通过电子邮件地址跟踪您参加测验的情况。请在整个测验中使用相同的电子邮件地址。

我们三个人共享一台计算机，我如果设置计算机以使我们都能参加测验？

为每个参加测验的人员设置不同的用户目录。缺省情况下，用户目录是一个名为 .netbeans 的隐藏目录，它存储在 .netbeans/[nb-version] 的用户 HOME 目录中。在类似 UNIX 的系统上，HOME 目录为 ${HOME}；在 Windows 上，HOME 目录为 C:\Documents 和 Settings\[user]。有关详细信息，请阅读常见问题解答什么是 .netbeans 目录？和我可以使用备用“用户目录”运行 NetBeans 吗？。

如何获得有关 NetBeans IDE 的一般帮助？

您可以阅读文档或 NetBeans 用户常见问题解答（尤其是本文档）。

每周是如何选择获奖者的？

在测验结束时，我们将生成一个列表，其中包含发送了所有问题的正确答案的所有参赛者，然后随机选择一名参赛者。请参见伪代码中的算法： 

      
 if (now == Algorithm.End_Date) {

   Array correct= find(users who answered correctly all questions in quiz)

   int winnerIdx = Random() // randomly chose index

   return winner = correct[winnerIndex]

 }

As implementation is used: java.util.Random 

      
int winnerIdx = new Random().nextInt(correct.size());

我如何知道每周的测验是否获奖？

将通过电子邮件通知所有获奖者，并且每周在测验获奖者页中发布获奖者名单。

我如何赢得大奖？

大奖将授予在每周测验中累计提交正确答案次数最多的人员。如果出现平分，将随机从这些参赛者中抽取大奖获奖者。

下周的奖品是什么？

奖品页中列出了奖品。

我可以在哪里看到获奖名单？

每周在测验获奖者页中发布获奖者名单。

有多少个测验？

11 个。测验从 2008 年 4 月 14 日开始，每周一次。

提交测验的截止日期是什么时间？

每周的测验从周一中午 12:01（印度标准时间）开始，到下周一中午 11:59（印度标准时间）结束。

我刚刚知道本次比赛，该比赛在两周前就开始了。我可以参加前几周的测验吗？

可以！测验插件将通知您前几周的测验。

如果别人已赢得了每周测验的奖品，我为什么还要参加旧测验呢？

大奖是根据所有测验的累计分数评选的。
