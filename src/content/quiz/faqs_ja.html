[国名] に住んでいます。参加できますか。

このコンテストは、エントリ時点で成年に達しているインドの合法的居住者のみを対象としています。詳細については、コンテスト規則を参照してください。

NetBeans IDE をすでに所有しています。クイズプラグインを入手するにはどうすればいいですか。

NetBeans IDE から直接クイズプラグインをインストールします。「ツール」>「プラグイン」に移動し、使用可能なプラグインの一覧で、クイズプラグインを見つけます。インストールするものを選択し、「インストール」ボタンをクリックします。

NetBeans アップデートセンターに接続できません。どうすればよいですか。

プロキシ設定を確認してください。「ツール」>「オプション」>「一般」に移動します。「プロキシなし」、「システムのプロキシ設定を使用」、「手動でプロキシ設定」のいずれかを選択して、IDE でインターネットへのアクセスに使用する一般プロキシ設定を指定します。「手動でプロキシ設定」を使用する場合は、HTTP プロキシおよびプロキシポートを指定する必要があります。HTTP プロキシは、HTTP サイトおよび FTP サイトへの接続に使用するプロキシサーバーのホスト名、または数値 IP アドレス番号です。プロキシポートは、プロキシサーバーのポート番号です。プロキシの詳細設定を設定するには、「詳細」をクリックします。

クイズにはどのバージョンの NetBeans IDE を使用できますか。

NetBeans IDE 5.5 以降 (NetBeans IDE 5.5、5.5.1、6.0、6.0.1、6.1 Beta、および 6.1 Release Candidate)。

クイズ中にオペレーティングシステムをアップグレードします。以前のクイズがすべて失われますか。

いいえ、新しいクイズプラグインをインストールしてもクイズのポイントを貯めることができます。電子メールアドレスでユーザーを追跡しています。クイズではずっと同じ電子メールアドレスを使用してください。

3 人で同じコンピュータを共有しています。全員がクイズに参加できるようにするにはどのように設定すればよいですか。

クイズに参加する人ごとに別のユーザーディレクトリを設定します。デフォルトで、ユーザーディレクトリはユーザーのホームディレクトリの .netbeans/[nb-version] の下に格納されている .netbeans という非表示ディレクトリです。ホームディレクトリは UNIX 系システムでは ${HOME}、Windows では C:\Documents and Settings\[user] です。詳細については、「.netbeans ディレクトリとは何ですか。」および「代替の「ユーザーディレクトリ」で NetBeans を実行できますか。」の FAQ を参照してください。

NetBeans IDE 全般のヘルプを取得するにはどうすればいいですか。

ドキュメントまたはNetBeans ユーザー FAQ（特にこの FAQ）をお読みください。

ウィークリー賞の当選者はどのように選ばれるのですか。

クイズが終了すると、すべての質問に正しい答えを送信したすべての参加者のリストを生成して、1 人の参加者を無作為に選出します。疑似コードのアルゴリズムを参照してください。 

      
 if (now == Algorithm.End_Date) {

   Array correct= find(クイズの全質問に正解したユーザー)

   int winnerIdx = Random() // ランダムにインデックスを選択
   return winner = correct[winnerIndex]

 }

実装が使われている: java.util.Random 

      
int winnerIdx = new Random().nextInt(correct.size());

ウィークリー賞を取るとどのように知らされるのですか。

すべての当選者は電子メールで通知され、当選者のリストが毎週クイズ当選者ページに掲載されます。

グランプリを取るにはどうすればいいのですか。

グランプリは、毎週のクイズからもっとも多く正しい答えを累積的に提出した人に与えられます。同点の場合、グランプリの当選者はトップの参加者から無作為に決定されます。

次週の賞品は何ですか。

賞品は賞品ページに掲載されます。

当選者のリストはどこで見られますか。

当選者のリストは、クイズ当選者ページに毎週掲載されます。

クイズは何回ありますか。

11 回です。クイズは 2008 年 4 月 14 日から毎週あります。

クイズの提出期限はいつですか。

毎週のクイズは月曜日の午後 12 時 1 分 (インド標準時) に開始され、次の月曜日の午後 11 時 59 分 (インド標準時) に終了します。

このコンテストに気付いたばかりで、数週間前に始まっていました。以前の週からクイズに参加できますか。

はい。クイズプラグインが、以前の週のクイズについて教えてくれます。

ほかの人がウィークリー賞を取っているのに、なぜ昔のクイズに参加するべきなのでしょうか。

グランプリはすべてのクイズの累積得点に基づくからです。
