Projetos Companheiros do NetBeans


Ao longo dos anos, o projeto NetBeans trabalhou de perto com outros projetos e comunidades de projetos de código-fonte aberto. Aprende sobre algumas destas parcerias e os benefícios de nossa colaboração para os desenvolvedores que planejam levar seus projetos para o próximo nível.

GlassFish

O GlassFish é o servidor de aplicativo de código-fonte aberto da Sun. Sua versão atual, GlassFish v3, foi reescrita para que seja modular, de peso leve e extensível. Ele foi desenvolvido sobre o OSGi e suporta o Java EE 6. Não somente aplicativos da Web Java podem ser implementados no mesmo: O Ruby e Groovy também são suportados. O NetBeans IDE suporta o GlassFish. Quando você cria um novo aplicativo Web Java, pode especificar que deseja que seja implementado no GlassFish. Você também pode ver os aplicativos implementados no GlassFish e os implementar através do IDE.

MySQL

O banco de dados MySQL se tornou o banco de dados de código-fonte aberto mais popular do mundo, por causa de seu desempenho rápido e consistente, alta confiabilidade e facilidade de uso. Ele é usado em cada continente por desenvolvedores individuais da Web, assim como por muitas das maiores organizações de crescimento rápido no mundo, para economizar tempo e dinheiro, dando poder para seus sites da Web de alto volume, sistemas comerciais crítico e software empacotado. O NetBeans IDE simplifica a criação de aplicativos que se conectam ao banco de dados MySQL, e permite criar, visualizar e alterar bancos de dados MySQL através do IDE.

OpenSolaris

O OpenSolaris é um sistema operacional (SO), um projeto de código-fonte aberto licenciado sob a CDDL, e uma comunidade. O OpenSolaris é grátis e bem adequado para desktops, laptops, servidores e centros de dados. O OpenSolaris tem recursos especiais que o distingue de outros sistemas operacionais. Por exemplo, ele inclui o DTrace, um utilitário que permite que os usuários e administradores examinem os aspectos internos do sistema operacional e modificar de forma dinâmica seu kernel. Com o suporte para este recurso, o NetBeans IDE fornece uma interface de usuário gráfica sob o DTrace, para que os usuários do NetBeans no OpenSolaris possam visualizar com facilidade a saída do DTrace.

OpenJDK

A Sun Microsystems estabeleceu a Comunidade OpenJDK para o desenvolvimento contínuo da implementação de código-fonte aberto do Java SE da Sun. A Comunidade OpenJDK é onde os desenvolvedores se juntam para colaborar na base do código-fonte aberto JDK e em projetos relacionados. Os Projetos OpenJDK 6 e 7 onde vive, a base do código-fonte aberto fazem parte da comunidade. Através dos Projetos OpenJDK, os desenvolvedores podem influenciar diretamente o futuro da implementação JDK, participar com seus pares em uma comunidade aberta e a ajudar a levar a tecnologia Java para onde ainda não esteve antes. O OpenJDK é tornado disponível como projetos NetBeans. Após baixá-los, você pode simplesmente abri-los em um IDE e trabalhar com eles da mesma forma que faria com outro projeto.

OpenOffice


 O OpenOffice.org é o conjunto de software de código fonte aberto para escritório líder para processamento de palavra, planilhas, apresentações, bancos de dados, e mais. Ele está disponível em muitos idiomas e funciona em todos os computadores comuns. Ele armazena seus dados em um formato padrão interno e também pode ler e gravar arquivos de outros pacotes comuns de software para escritório. Ele pode ser baixado e usado completamente de graça para qualquer propósito. Mais ainda, você pode usar o plug-in API do OpenOffice.org API para o NetBeans IDE para simplificar o desenvolvimento de extensões e aplicativos cliente do OpenOffice.

VirtualBox

O VirtualBox é uma família de poderosos produtos de virtualização x86 para corporações e para uso doméstico. O VirtualBox tem recursos extremamente ricos, é um produto de alto desempenho para clientes corporativos, mas também é uma solução profissional que é livremente grátis como um Software de Código- fonte Aberto sob os termos da Licença Pública Geral GNU(GPL). O VirtualBox está sendo ativamente desenvolvido com frequentes versões e tem uma crescente lista de recursos, sistemas operacionais convidados suportados e uma plataforma na qual é executado.
