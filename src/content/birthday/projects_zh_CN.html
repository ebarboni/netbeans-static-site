NetBeans 配套项目


多年以来，NetBeans 项目与其他开源项目和社区配合使用。请了解这些合作伙伴关系以及这种合作给寻求完成项目开发的开发者带来的好处。

GlassFish

GlassFish 是 Sun 的开源应用服务器。目前的版本 GlassFish v3 已重新进行编写，以使其具有模块化、小型和可扩展的特点。它是在 OSGi 上构建的，并支持 Java EE 6。不仅可以在上面部署 Java Web 应用程序，而且还支持 Ruby 和 Groovy。NetBeans IDE 支持 GlassFish。在创建新的 Java Web 应用程序时，您可以指定将其部署到 GlassFish 上。您还可以查看部署到 GlassFish 上的应用程序，以及从 IDE 中取消部署这些应用程序。

MySQL

MySQL 数据库由于始终具有速度快、可靠性高和简便易用性的优点，已成为世界上最流行的开源数据库。个人 Web 开发者以及世界上很多最大和增长速度最快的组织都纷纷使用该数据库，以便在支持其高容量的网站、业务关键系统和打包的软件方面节省大量的时间和资金。NetBeans IDE 简化了创建连接到 MySQL 数据库的应用程序的过程，并允许您在 IDE 中创建、查看和更改 MySQL 数据库。

OpenSolaris

OpenSolaris 是一个操作系统 (OS)、依照 CDDL 许可的开源项目以及社区。OpenSolaris 是免费提供的，非常适合台式机、笔记本、服务器和数据中心。OpenSolaris 包含一些特殊功能，这是其他操作系统所没有的。例如，它包含 DTrace 实用程序，用户和管理员可用来检查操作系统的内部结构并动态地修改其内核。NetBeans IDE 在 DTrace 上提供图形用户界面以支持此功能，以使 OpenSolaris 上的 NetBeans 用户可以轻松可视化 DTrace 输出。

OpenJDK

Sun Microsystems 建立了 OpenJDK 社区以支持 Sun 持续开发 Java SE 开源实现。OpenJDK 社区为开发者提供了一个地方，他们聚在一起合作完成开源 JDK 代码库和相关的项目。开源代码库所在的 OpenJDK 6 和 7 项目是此社区的一部分。通过 OpenJDK 项目，开发者可以直接影响 JDK 实现的未来发展方向，与他们的同行参与开放的社区，以及帮助宣传推广该 Java 技术。OpenJDK 是作为 NetBeans 项目提供的。在下载这些项目后，您只需在 IDE 中打开这些项目，然后便可像任何其他项目一样使用这些项目。

OpenOffice


 OpenOffice.org 是领先的开源办公软件套件，其中包括文字处理、电子表格、演示文稿、图形和数据库等。它提供了多种语言版本，并且可以在所有常见的计算机上使用。它使用国际开放标准格式存储所有数据，并且还可以在其他常见的办公软件包中读写文件。可以完全免费下载和使用该套件以用于任何目的。此外，您还可以使用 NetBeans IDE 的 OpenOffice.org API 插件简化 OpenOffice 扩展和客户端应用程序开发。

VirtualBox

VirtualBox 是一个为企业和家庭用户提供的强大 x86 虚拟化产品系列。VirtualBox 不仅是为企业客户提供的功能极其丰富的高性能产品，而且也是唯一依照 GNU 通用公共许可证 (GPL) 条款作为开源软件免费提供的专业解决方案。VirtualBox 经常发布开发的新版本，并且不断增添提供的功能、支持的来宾操作系统和运行它的平台。
