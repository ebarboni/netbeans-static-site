NetBeans 関連プロジェクト


長年にわたって、NetBeans プロジェクトはほかのオープンソースプロジェクトおよびコミュニティーと協力して取り組んできました。これらのパートナーシップのいくつかについてと、次の段階へのプロジェクトのレベルアップを目指している開発者との共同作業のメリットについて学んでください。

GlassFish

GlassFish とは Sun のオープンソース化されたアプリケーションサーバーです。最新バージョンの GlassFish v3 は、モジュール化され、軽量で、拡張可能になるように書き換えられています。それは OSGi 上に構築され、Java EE 6 をサポートします。それに配備できるのは Java Web アプリケーションだけではありません。Ruby や Groovy もサポートされます。NetBeans IDE は GlassFish をサポートしています。Java Web アプリケーションを新たに作成するときに、それが GlassFish に配備されるように指定できます。また、GlassFish に配備されているアプリケーションを表示して、IDE 内からそれらの配備を取り消すこともできます。

MySQL

MySQL データベースは、その一貫した高速性能、高信頼性、および使いやすさのために世界でもっとも普及しているオープンソースデータベースになりました。あらゆる大陸で、世界で最大かつ急速に成長している組織だけでなく個々の Web 開発者も、大容量の Web サイト、ビジネスクリティカルなシステム、およびパッケージ化されたソフトウェアを強力なものにする時間とお金を節約するためにそれを使用します。NetBeans IDE は、MySQL データベースに接続するアプリケーションの作成を簡素化し、IDE 内から MySQL データベースを作成、表示、および変更できるようにします。

OpenSolaris

OpenSolaris とは、オペレーティングシステム (OS)、CDDL に基づいてライセンスされたオープンソースプロジェクト、およびコミュニティーです。OpenSolaris は無償であり、デスクトップ、ノートパソコン、サーバー、およびデータセンターによく適しています。OpenSolaris には、それをほかのオペレーティングシステムと区別する特別な機能があります。たとえば、それには DTrace という、ユーザーや管理者がオペレーティングシステムの内部を調査し、そのカーネルを動的に変更できるようにするユーティリティーが含まれています。この機能をサポートすることで、NetBeans IDE は DTrace 上にグラフィカルユーザーインタフェースを備えて、OpenSolaris 上の NetBeans ユーザーが DTrace の出力を簡単に図形化できるようにします。

OpenJDK

Sun Microsystems は、Sun による Java SE のオープンソース実装の進行中の開発のために OpenJDK コミュニティーを設立しました。OpenJDK コミュニティーは、開発者が集まってオープンソースの JDK コードベースや関連プロジェクトに共同で取り組む場所です。オープンソースのコードベースが有効な OpenJDK 6 および 7 プロジェクトは、このコミュニティーの一部です。OpenJDK プロジェクトを通して、開発者は JDK 実装の将来に直接影響を与えたり、オープンコミュニティーに仲間とともに参加したり、以前はなかった場所に Java テクノロジを持っていくのを手伝ったりできます。OpenJDK は NetBeans プロジェクトとして利用できるようになりました。それらのダウンロードが完了したら、IDE でそれらを開くだけで、ほかのプロジェクトと同様にそれらを使って作業できます。

OpenOffice


 OpenOffice.org は、ワードプロセッシング、スプレッドシート、プレゼンテーション、グラフィックス、データベースその他に適した主要なオープンソースのオフィスソフトウェアです。それは多くの言語で使用でき、一般的なすべてのコンピュータで動作します。それはすべてのデータを国際的なオープン標準形式で保存し、ほかの一般的なオフィスソフトウェアパッケージからのファイルの読み取り/書き込みも可能です。それはどのような目的でも完全に無償でダウンロードおよび使用できます。さらに、NetBeans IDE に対して OpenOffice.org API プラグインを使用すると、OpenOffice 拡張とクライアントアプリケーションの開発を簡素化できます。

VirtualBox

VirtualBox とは、ホームだけでなくエンタープライズでの使用にも適した強力な x86 仮想化製品ファミリです。VirtualBox は、エンタープライズカスタマ向けの非常に機能豊富で高性能な製品であるだけでなく、GNU GPL (General Public License) の条項に基づいてオープンソースソフトウェアとして自由に利用できるプロフェッショナルソリューションでもあります。VirtualBox は頻繁なリリースととも積極的に開発されており、機能、サポートされるゲストオペレーティングシステム、およびそれが動作するプラットフォームの増え続けるリストを備えています。
