Дополняющие проекты NetBeans


В течение многих лет проект NetBeans развивался совместно с другим проектами с открытым исходным кодом и различными сообществами. В этом разделе приведена информация о партнерствах и преимуществах сотрудничества для разработчиков, желающих реализовывать проекты на новом уровне.

GlassFish

GlassFish - это сервер приложений Sun с открытым исходным кодом. Текущая версия GlassFish v3 стала более универсальной, легкой и расширяемой. Она построена на основе OSGi и поддерживает Java EE 6. На ней поддерживаются не только веб-приложения Java, но также Ruby и Groovy. Среда IDE NetBeans также поддерживает GlassFish. При создании нового веб-приложения Java можно выбрать его развертывание на GlassFish. Также можно просматривать приложения, реализованные на GlassFish, и отменять их развертывание в среде IDE.

MySQL

Благодаря стабильно высокой производительности, надежности и удобстве использования, база данных MySQL является самой популярной в мире базой данных с открытым исходным кодом. На всех континентах она используется как самостоятельными веб-разработчиками, так и крупнейшими и наиболее динамичными компаниями, позволяя им экономить время и деньги, поддерживая крупные веб-сайты, критически важные для бизнеса системы и пакетное программное обеспечение. IDE NetBeans упрощает создание приложений, использующих базы данных MySQL, и позволяет создавать, просматривать и изменять их в среде IDE.

OpenSolaris

OpenSolaris - это операционная система, проект с открытым кодом, лицензированный по CDDl, а также сообщество. OpenSolaris распространяется бесплатно и подходит для настольных компьютеров, ноутбуков, серверов и центров обработки данных. OpenSolaris имеет особые функции, отличающие ее от других операционных систем. Например, она содержит служебную программу DTrace, позволяющую пользователям и администраторам анализировать внутренние компоненты операционной системы и динамически изменять ее ядро. Поддерживая эту функцию, среда IDE NetBeans обеспечивает графический интерфейс пользователя на основе DTrace для удобной визуализации DTrace пользователями NetBeans в OpenSolaris.

OpenJDK

Sun Microsystems создала сообщество OpenJDK для текущей разработки реализации Java SE с открытым исходном кодом Sun. Сообщество OpenJDK объединяет разработчиков, участвующих в создании открытого базового кода JDK и связанных с ним проектах. Это сообщество занимается проектами OpenJDK 6 и 7, в которых используется открытый базовый код. В проектах OpenJDK разработчики могут непосредственно влиять на будущую реализацию JDK, участвовать в открытом сообществе и применять технологии Java в тех областях, где они до сих пор не использовались. Проекты OpenJDK доступны как проекты NetBeans. После загрузки достаточно открыть их в среде IDE и работать с ними, как с любыми другими проектами.

OpenOffice


 OpenOffice.org - это наиболее востребованный пакет офисного программного обеспечения для работы с текстами, электронными таблицами, презентациями, рисунками, базами данных и т.д. Он доступен на многих языках и может использоваться на всех стандартных компьютерах. Данные сохраняются в стандартном открытом международном формате. Файлы могут открываться из других распространенных пакетов офисного программного обеспечения. OpenOffice загружается и используется бесплатно и в любых целях. Кроме того, в среде IDE NetBeans с помощью подключаемого модуля API OpenOffice.org можно упростить разработку расширений и клиентских приложений OpenOffice.

VirtualBox

VirtualBox - это семейство мощных продуктов для виртуализации x86 как для корпоративных, так и для частных пользователей. VirtualBox является не только многофункциональным и высокопроизводительным продуктом для корпоративных клиентам, но и единственным профессиональным решением, свободно распространяемым как программное обеспечение с открытым кодом в соответствии с GNU General Public License (GPL). VirtualBox активно развивается через частый выпуск новых версий и имеет постоянно расширяемую функциональность, список поддерживаемых операционных систем и платформ.
